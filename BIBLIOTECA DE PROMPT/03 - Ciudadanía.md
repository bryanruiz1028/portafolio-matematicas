# Prompts — Educación para la Ciudadanía

tags: #prompt #ciudadania #bachillerato #ciencias-sociales

## Prompt base

```
Actúa como un profesor experto en la asignatura de Educación para la Ciudadanía
del subnivel de Bachillerato.

Tema: [TEMA]
Destreza: [DESTREZA DEL CURRÍCULO]

[TAREA ESPECÍFICA]
```

---

## Variantes del prompt base

### Para generar material de clase

```
Actúa como un profesor experto en la asignatura de Educación para la Ciudadanía
del subnivel de Bachillerato.

Tema: [TEMA]

Genera dos documentos:
- HOJA DEL DOCENTE: metodología, preguntas guía, criterios de evaluación
- HOJA DEL ESTUDIANTE: actividades, preguntas de análisis, reflexión crítica
```

### Para análisis de temas sociales y políticos

```
Actúa como un profesor experto en Educación para la Ciudadanía (Bachillerato).

Analiza el siguiente tema desde una perspectiva pedagógica equilibrada:
[TEMA — ej: democracia, derechos humanos, participación ciudadana]

Incluye:
1. Conceptos clave para trabajar en clase
2. Perspectivas distintas sobre el tema
3. Preguntas para fomentar el pensamiento crítico
4. Actividad participativa sugerida
```

---

## Temas trabajados

- Origen y evolución de la democracia en Ecuador
- Izquierda y Derecha (ideologías políticas)
- Demandas sociales y construcción del Estado incluyente
- VIH/SIDA: acceso a apoyos médicos y sociales

---

## Notas

- Este prompt funciona bien combinado con [[01 - Generador de material de clase]]
- Para temas sensibles (salud, política), especificar enfoque pedagógico neutro
