# Planificaciones curriculares

tags: #prompt #planificacion #curriculum #evaluacion

## Prompts de esta sección

### 1. Prompt con el tema

```
Actúa como un experto en planificación curricular ecuatoriana.

Asignatura: [MATERIA]
Subnivel: [ELEMENTAL / MEDIA / SUPERIOR / BACHILLERATO]
Tema: [TEMA DE LA UNIDAD]

Genera una planificación de clase con:
- Objetivo de aprendizaje
- Destrezas con criterios de desempeño
- Estrategias metodológicas
- Recursos
- Indicadores de evaluación
```

---

### 2. Destrezas e indicadores

```
Actúa como un experto en currículo ecuatoriano.

Para la asignatura de [MATERIA], subnivel [NIVEL], tema [TEMA]:
Lista las destrezas con criterios de desempeño correspondientes
y sus indicadores de evaluación según el currículo nacional.
```

---

### 3. Temas de las destrezas

```
Actúa como un docente experto en [MATERIA].

Dado el siguiente listado de destrezas del currículo:
[PEGAR DESTREZAS]

Sugiere los temas específicos de clase que permiten desarrollar
cada destreza, ordenados de menor a mayor complejidad.
```

---

### 4. Creación de examen

```
Actúa como un evaluador educativo experto.

Asignatura: [MATERIA]
Nivel: [NIVEL]
Tema(s): [TEMAS]
Tipo de preguntas: [opción múltiple / verdadero-falso / abiertas / mixto]
Número de preguntas: [N]

Crea un examen con clave de respuestas al final.
```
