## Estructura de un prompt efectivo

1. **Rol** — define quién es la IA (`Actúa como un experto en...`)
2. **Contexto** — materia, nivel, subnivel educativo
3. **Tarea** — qué debe producir exactamente
4. **Salida esperada** — formato, documentos, extensión
