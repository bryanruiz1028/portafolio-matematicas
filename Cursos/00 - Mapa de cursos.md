## 🗺️ Ruta de aprendizaje — De menor a mayor complejidad

> [!tip] Sigue este orden para una progresión pedagógica óptima
> La ruta va de fundamentos numéricos hasta cálculo avanzado y física.

| Nivel | Área                   | Cursos clave                                                                                                  |
| ----- | ---------------------- | ------------------------------------------------------------------------------------------------------------- |

| 🟢 2  | Geometría básica       | Ángulos → Polígonos → Triángulos → Área y Perímetro → Teorema de Pitágoras → Teorema de Tales                 |
| 🟡 3  | Álgebra inicial        | Conjuntos → Lenguaje Algebraico → Expresiones → Propiedades de Potenciación → Razones y proporciones          |

| 🟡 5  | Números avanzados      | Irracionales/Radicales → Números Complejos → Sucesiones → Logaritmos                                          |
| 🟠 6  | Estadística            | Conceptos básicos → Gráficos → Medidas centrales → Tabla de frecuencias → Varianza → Probabilidad             |
| 🟠 7  | Geometría Analítica    | Plano Cartesiano → Recta → Circunferencia → Parábola → Elipse → Hipérbola                                     |

| 🔴 9  | Álgebra avanzada       | Funciones → Función compuesta → Ecuaciones exponenciales → Matrices                                           |
