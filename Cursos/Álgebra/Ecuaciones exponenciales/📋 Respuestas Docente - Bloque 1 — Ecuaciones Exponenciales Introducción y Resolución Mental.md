**Clave de Respuestas (Q1-Q8)**

*   **Q1:** **c) 6**. *Justificación:* Siguiendo la recomendación del Profe Alex de conocer las potencias de memoria, $2$ multiplicado por sí mismo $6$ veces es $64$ ($2^6 = 64$).
*   **Q2:** **b) Los exponentes son iguales**. *Justificación:* Al tener una sola base igual a cada lado ($a^m = a^n$), los exponentes deben equilibrarse por propiedad.
*   **Q3:** **b) $3^3$**. *Justificación:* Se requiere convertir el 27 a base 3 para poder aplicar la igualdad de exponentes.
*   **Q4:** **Verdadero**. *Justificación:* Es la propiedad fundamental del exponente cero explicada en la introducción.
*   **Q5:** **Falso**. *Justificación:* El contexto enfatiza que solo funciona si hay un solo término (base) a cada lado de la igualdad.
*   **Q6:** bases / ecuación / exponentes.
*   **Q7:** ecuación / variable / potencias.
*   **Q8 (Relación):**
    1. $5^x = 25 \implies x = 2$
    2. $2^x = 8 \implies x = 3$
    3. $3^x = 81 \implies x = 4$
    4. $10^x = 100 \implies x = 2$

**Procedimientos Detallados**

*   **Resolución Q9:**
    1. Las bases ya son iguales (base 5).
    2. Se igualan los exponentes: $x + 2 = 7$.
    3. Se despeja $x$: $x = 7 - 2$.
    4. Resultado: $x = 5$.
*   **Resolución Q10:**
    1. Se reconoce que 32 es una potencia de 2: $2^5 = 32$.
    2. Se sustituye en la ecuación: $2^x = 2^5$.
    3. Por propiedad de bases iguales: $x = 5$.
    4. Resultado: Deben pasar 5 días.
