### 1. Clave de Respuestas Detallada

*   **Q1: b) $3xy$.** Se extrae el $3$ como MCD de $9$ y $12$. En la parte literal, se eligen $x$ y $y$ con su exponente más pequeño ($1$).
*   **Q2: c) $5m^2$.** Basado en el "Profe Alex-ism": el factor común literal es siempre la letra que se repite con el exponente menor.
*   **Q3: d) $6$.** El divisor más grande para $12$ y $18$ es $6$.
*   **Q4: V.** Definición fundamental: la factorización es la operación inversa a la multiplicación.
*   **Q5: F.** Error didáctico común. Se debe usar el exponente *más pequeño* para evitar exponentes negativos al dividir.
*   **Q6: c) dividir / a) término.** Proceso de descomposición de la suma original.
*   **Q7: b) simplificar / d) factor.** Justificación técnica: Al dividir $5m^2 / 5m^2$, el resultado es $1$. Es un error frecuente de los estudiantes colocar $0$; el $1$ debe permanecer como factor para que la multiplicación inversa sea posible.
*   **Q8 (Relación):** 1-C, 2-D, 3-A, 4-B.

### 2. Resolución de Problemas (Q9 y Q10)

**Q9. Expresión $5m^2 + 15m^3$:**
1.  **MCD:** $5$. **Literal:** $m^2$. **Factor Común:** $5m^2$.
2.  **División:** $(5m^2 / 5m^2) = 1$ y $(15m^3 / 5m^2) = 3m$.
3.  **Resultado:** $5m^2(1 + 3m)$.
4.  **Verificación:** $5m^2 \cdot 1 = 5m^2$; $5m^2 \cdot 3m = 15m^3$.

**Q10. Aplicación USD ($12x^2 + 18x$):**
1.  **MCD:** $6$. **Literal:** $x$. **Factor Común:** $6x$.
2.  **División:** $(12x^2 / 6x) = 2x$ y $(18x / 6x) = 3$.
3.  **Resultado final:** $6x(2x + 3)$ USD.
4.  **Verificación:** $6x \cdot 2x = 12x^2$; $6x \cdot 3 = 18x$.

### 3. Rúbrica de Evaluación

| Sección | Logrado (100%) | En Proceso (50%) | Por Lograr (0%) |
| :--- | :--- | :--- | :--- |
| **Identificación de Factores** | Identifica el MCD y las variables de menor exponente sin error. | Identifica solo el factor literal o solo el numérico, o confunde el exponente menor con el mayor. | No identifica factores comunes o elige términos al azar. |
| **División Algebraica** | Divide cada término correctamente, manteniendo el coeficiente $1$ cuando es necesario. | Comete errores al restar exponentes o ignora el coeficiente $1$ al dividir términos idénticos (pone $0$). | No realiza la división o los resultados son incoherentes con el factor extraído. |
| **Verificación Técnica** | Comprende y aplica la multiplicación inversa para validar el resultado. | Plantea la factorización pero no intenta verificar o no detecta errores en el proceso inverso. | No establece relación entre la multiplicación y la factorización. |
