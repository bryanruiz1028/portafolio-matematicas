# 📖 Guía de estudio — Clase 02: Multiplicación de expresiones algebraicas

> [!info] 📌 Conceptos clave
> Para dominar la multiplicación algebraica según el método del Profe Alex, es fundamental integrar estos cuatro pilares:
> - **Monomio vs. Polinomio:** Un monomio es un solo término (ej. $5x^3$). Un polinomio tiene dos o más términos conectados por sumas o restas (ej. $3x + 2$).
> - **Regla de los exponentes:** En la multiplicación de bases iguales, la base se mantiene y los exponentes se suman ($x^a \cdot x^b = x^{a+b}$). 
>   - **⚠️ Nota vital:** Los coeficientes se **multiplican**, pero los exponentes se **suman**. ¡No los confundas!
> - **Orden secuencial (El "Paso a Paso"):** Para evitar errores, opera siempre en este orden: 1. **Signos**, 2. **Coeficientes** (números), 3. **Letras**. 
>   - **Prioridad de signos:** Si el resultado inicial es positivo, el signo es opcional; si es negativo, es **obligatorio**.
> - **Propiedad distributiva:** Al multiplicar un monomio por un polinomio, el monomio se multiplica por cada término interno. Generalmente, al terminar no habrá "términos semejantes" para simplificar, por lo que el proceso suele concluir ahí.

## Tabla de fórmulas y definiciones importantes

| Término | Definición / Fórmula |
| :--- | :--- |
| **Multiplicación de signos** | Signos iguales = (+). Signos diferentes = (-). |
| **Leyes de exponentes** | $\displaystyle x^a \cdot x^b = x^{a+b}$. Aplica también para negativos: $x^5 \cdot x^{-2} = x^{5-2} = x^3$. |
| **Exponentes e "Invisibles"** | Si una letra no tiene exponente o un término no tiene coeficiente, su valor es **1**. |
| **Multiplicación de fracciones** | $\displaystyle \frac{a}{b} \cdot \frac{c}{d} = \frac{a \cdot c}{b \cdot d}$. Multiplica horizontalmente y simplifica al final. |
| **Estrategia para Enteros** | Al multiplicar un entero por una fracción, coloca un **1** como denominador (ej. $3 = \frac{3}{1}$) para no errar. |

## Ejemplos resueltos adicionales

```ad-example
TITLE: Ejemplo A — Caso Básico (Monomio por Monomio)
**Ejercicio:** Resolver $(-6m^3) \cdot (-3m^5)$

**Paso a paso:**
1. **Multiplicación de signos:** Negativo por negativo resulta en positivo (+). No es necesario escribirlo al inicio.
2. **Multiplicación de coeficientes:** $6 \cdot 3 = 18$.
3. **Suma de exponentes:** La base es $m$. Sumamos $3 + 5 = 8$.

**Resultado final:** $18m^8$
```

```ad-example
TITLE: Ejemplo B — Aplicación real con $USD
**Problema:** Una empresa compra $5x^2$ artículos. Si cada artículo cuesta $3x^5$ USD, ¿cuál es la expresión del gasto total?

**Paso a paso:**
1. **Multiplicación de signos:** (+) por (+) resulta en positivo.
2. **Multiplicación de coeficientes:** $5 \cdot 3 = 15$.
3. **Suma de exponentes:** La base es $x$. Sumamos $2 + 5 = 7$.

**Resultado final:** El costo total es $15x^7$ USD.
```

