## Resolución Detallada Q9
1.  **Aplicar productos cruzados:** $2(4x + 1) = 3(4x - 1)$
2.  **Propiedad distributiva:** $2(4x) + 2(1) = 3(4x) - 3(1) \rightarrow 8x + 2 = 12x - 3$
3.  **Transposición de términos:** $8x - 12x = -3 - 2$
4.  **Reducción de términos semejantes:** $-4x = -5$
5.  **Regla del Profe Alex:** Como la $x$ está acompañada de un número negativo, multiplicamos toda la ecuación por $-1$: $4x = 5$
6.  **Despeje final:** $x = \frac{5}{4}$ (Al verificar, $4(5/4) \pm 1$ no es cero, la solución es válida).

## Resolución Detallada Q10
1.  **Planteamiento:** $\frac{60}{x} = 12$
2.  **Resolución:** $60 = 12x \rightarrow x = \frac{60}{12}$
3.  **Resultado:** $x = 5$.
4.  **Verificación de Indeterminación:** Si $x = 5$, el denominador es $5$ (diferente de cero).
**Respuesta final:** 5 empleados.
