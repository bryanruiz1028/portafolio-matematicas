**Q9 (Números consecutivos):**
1.  **Dar nombre:** $n$ (número menor), $n+1$ (número mayor).
2.  **Escribir ecuación:** $n + (n+1) = 451 \rightarrow 2n + 1 = 451$.
3.  **Resolver:** $2n = 450 \rightarrow n = 225$.
4.  **Verificar:** $225 + 226 = 451$. ¡Correcto! Los números son **225 y 226**.

**Q10 (Edades hermanos):**
1.  **Dar nombre:** $x$ (edad del menor), $x+4$ (edad del mayor).
2.  **Escribir ecuación:** $x + (x+4) = 34 \rightarrow 2x + 4 = 34$.
3.  **Resolver:** $2x = 30 \rightarrow x = 15$.
4.  **Verificar:** $15 + 19 = 34$ y la diferencia es $4$. ¡Correcto! El menor tiene **15 años** y el mayor **19 años**.

**Q11 (Lote rectangular):**
1.  **Dar nombre:** Ancho = $a$; Largo = $2a + 5$.
2.  **Escribir ecuación:** $a + a + (2a+5) + (2a+5) = 94 \rightarrow 6a + 10 = 94$.
3.  **Resolver:** $6a = 84 \rightarrow a = 14$.
4.  **Verificar:** Perímetro = $14+14+33+33 = 94$. ¡Correcto! **Ancho: 14m, Largo: 33m**.

**Q12 (Euros):**
1.  **Identificar/Nombre:** Camisa = $c$; Pantalón = $c+5$ (se le asigna la letra al menor costo).
2.  **Escribir ecuación:** $6(c+5) + 4c = 480$.
3.  **Resolver:** $6c + 30 + 4c = 480 \rightarrow 10c = 450 \rightarrow c = 45$.
4.  **Verificar:** $6(50) + 4(45) = 300 + 180 = 480$. ¡Correcto! **Camisa: 45€, Pantalón: 50€**.
