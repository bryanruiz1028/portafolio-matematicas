# Guía de Estudio: Resolución de Problemas con Ecuaciones de Primer Grado

Esta guía proporciona las herramientas necesarias para comprender y resolver problemas matemáticos mediante el uso de ecuaciones de primer grado. Se enfoca en la traducción del lenguaje común al lenguaje algebraico, el proceso de resolución paso a paso y la verificación de resultados.

---

## 1. Conceptos Clave

Para resolver problemas mediante ecuaciones, es fundamental dominar los siguientes pilares:

*   **Lenguaje Algebraico:** Es la capacidad de traducir situaciones de la vida real (palabras) a expresiones matemáticas (números y letras). Por ejemplo, "el doble de un valor" se traduce como $2x$.
*   **Identificación de la Incógnita:** Generalmente se asigna una letra (como $x$, $n$, $c$, o $h$) al valor desconocido. Una recomendación práctica es asignar la letra al valor que sea menor o más pequeño en el problema para facilitar las operaciones.
*   **Propiedad Distributiva:** Al resolver ecuaciones con paréntesis, se debe multiplicar el número de afuera por cada uno de los términos dentro del paréntesis (ej. $4(2n + 10) = 8n + 40$).
*   **Suma de Ángulos Internos:** En cualquier triángulo, la suma de sus tres ángulos internos siempre es igual a 180 grados.
*   **Perímetro:** Es la suma de todos los lados de una figura geométrica. En un triángulo isósceles, esto incluye dos lados iguales y una base.
*   **Verificación:** Es el proceso final de comprobar que los resultados obtenidos cumplen con todas las condiciones originales del enunciado.

---

## 2. Metodología de Resolución (Paso a Paso)

El método sugerido para abordar cualquier problema de ecuaciones consta de cuatro pasos principales:

1.  **Nombrar las incógnitas:** Elegir una letra para representar lo que se pregunta. Si hay varias preguntas, relacionarlas todas usando la misma letra.
2.  **Escribir la ecuación:** Traducir la relación entre los datos del problema a una igualdad matemática.
3.  **Resolver la ecuación:** Despejar la letra realizando las operaciones aritméticas necesarias (agrupar términos semejantes, pasar números a un lado y letras al otro).
4.  **Verificar y dar la respuesta:** Comprobar si el resultado tiene sentido con el enunciado y redactar la solución final.

---

## 3. Ejemplos Resueltos Paso a Paso

### Ejemplo A: Problema de Compras (Pantalones y Camisas)
**Enunciado:** Se compraron 6 pantalones y 4 camisas por 480 euros. Si cada pantalón costó 5 euros más que cada camisa, ¿cuánto valió cada prenda?

*   **Paso 1: Nombrar las incógnitas**
    *   Valor de una camisa: $c$ (es el menor valor).
    *   Valor de un pantalón: $c + 5$ (cuesta 5 más que la camisa).
*   **Paso 2: Escribir la ecuación**
    Multiplicamos la cantidad por el precio unitario:
    $6(c + 5) + 4c = 480$
*   **Paso 3: Resolver**
    1.  Aplicar distributiva: $6c + 30 + 4c = 480$
    2.  Agrupar letras: $10c + 30 = 480$
    3.  Despejar: $10c = 480 - 30 \Rightarrow 10c = 450$
    4.  Dividir: $c = 450 / 10 \Rightarrow c = 45$
*   **Paso 4: Respuesta y Verificación**
    *   Camisa: 45 euros.
    *   Pantalón: $45 + 5 = 50$ euros.
    *   *Verificación:* $6(50) + 4(45) = 300 + 180 = 480$. Es correcto.

### Ejemplo B: Problema de Geometría (Ángulos de un Triángulo)
**Enunciado:** Hallar los tres ángulos de un triángulo sabiendo que el ángulo $A$ mide el doble que $B$, y el ángulo $C$ mide 20 grados menos que $B$.

*   **Paso 1: Nombrar las incógnitas**
    *   Ángulo $B$: $b$
    *   Ángulo $A$: $2b$
    *   Ángulo $C$: $b - 20$
*   **Paso 2: Escribir la ecuación**
    La suma de los ángulos de un triángulo es 180:
    $b + 2b + (b - 20) = 180$
*   **Paso 3: Resolver**
    1.  Agrupar términos semejantes: $4b - 20 = 180$
    2.  Pasar el 20 a sumar: $4b = 180 + 20 \Rightarrow 4b = 200$
    3.  Dividir: $b = 200 / 4 \Rightarrow b = 50$
*   **Paso 4: Respuesta**
    *   Ángulo $B = 50^\circ$.
    *   Ángulo $A = 100^\circ$ (el doble de 50).
    *   Ángulo $C = 30^\circ$ ($50 - 20$).

## 6. Glosario de Términos Importantes

| Término | Definición |
| :--- | :--- |
| **Denominación** | El valor específico de una moneda o billete (ej. monedas de 50 pesos o de 100 pesos). |
| **Ecuación de Primer Grado** | Una igualdad matemática donde la incógnita (variable) tiene un exponente de 1. |
| **Incógnita** | El valor desconocido que se busca encontrar en un problema, representado comúnmente por una letra. |
| **Lenguaje Braile (Contexto)** | Término coloquial usado por el autor para referirse a la asignación de nombres o etiquetas a las variables antes de operar. |
| **Perímetro** | La medida del contorno de una figura; se obtiene sumando la longitud de todos sus lados. |
| **Términos Semejantes** | Expresiones que tienen la misma parte literal (la misma letra), lo que permite sumarlas o restarlas entre sí (ej. $6c$ y $4c$). |
| **Triángulo Isósceles** | Tipo de triángulo que se caracteriza por tener dos lados iguales y uno diferente (base). |
| **Verificación** | Proceso de sustituir los valores encontrados en las condiciones originales del problema para asegurar que la igualdad se cumpla. |
