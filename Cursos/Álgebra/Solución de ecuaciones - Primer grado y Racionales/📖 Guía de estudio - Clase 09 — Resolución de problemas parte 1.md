# Guía de Estudio: Resolución de Problemas con Ecuaciones de Primer Grado

Esta guía está diseñada para estudiantes de nivel Básica Superior. Su objetivo es facilitar la comprensión de cómo transformar problemas de la vida cotidiana y acertijos numéricos en ecuaciones matemáticas para encontrar soluciones de forma lógica y estructurada.

---

## 1. Conceptos Clave y Lenguaje Algebraico

Para resolver problemas con ecuaciones, el primer paso fundamental es traducir palabras a "lenguaje algebraico" (números y letras).

### Traducciones comunes
| Frase en lenguaje común | Expresión Algebraica |
| :--- | :--- |
| Un número cualquiera | $n$ (o cualquier letra como $x, a, b$) |
| El doble de un número | $2n$ |
| El triple de un número | $3n$ |
| El quíntuple de un número | $5n$ |
| La mitad de un número | $n / 2$ |
| La quinta parte de un número | $n / 5$ |
| Un número aumentado en... | $n + \dots$ |
| Un número disminuido en... | $n - \dots$ |
| La diferencia | Indica una resta ($-$) |
| Se obtiene / Equivale a / Es igual a | Indica el signo igual ($=$) |

### Relaciones entre números
*   **Números consecutivos:** Son números enteros que van uno tras otro (ej. 4 y 5). Se representan como $n$ (el menor) y $n + 1$ (el siguiente). Si son tres, se añade $n + 2$.
*   **Sucesor:** Es el número que sigue ($n + 1$).
*   **Antecesor:** Es el número anterior ($n - 1$).

## 2. Pasos para Resolver un Problema con Ecuaciones

El análisis de los problemas debe seguir un orden lógico para evitar errores:

1.  **Identificar la pregunta y dar nombre a la variable:** Determinar qué se busca y asignarle una letra (por ejemplo, $n$ para "número" o $a$ para "años").
2.  **Escribir la ecuación:** Traducir el enunciado del problema al lenguaje algebraico.
3.  **Resolver la ecuación:**
    *   Si hay fracciones, multiplicar toda la ecuación por el denominador para eliminarlas.
    *   Agrupar los términos con la letra a un lado de la igualdad y los números al otro (recordando cambiar el signo al mover términos).
    *   Realizar las operaciones y despejar la letra.
4.  **Verificar el resultado:** Sustituir el valor encontrado en el enunciado original para comprobar si se cumplen las condiciones por lógica.
5.  **Dar la respuesta con palabras:** Responder formalmente a la pregunta planteada.

## 3. Ejemplos Resueltos Paso a Paso

### Ejemplo A: Números y Fracciones
**Problema:** El doble de un número y su mitad suman 45. ¿Cuál es el número?

*   **Paso 1 (Variable):** $n$ es el número.
*   **Paso 2 (Ecuación):** $2n + \frac{n}{2} = 45$
*   **Paso 3 (Resolución):**
    1.  Multiplicamos todo por 2 para quitar la fracción: $2(2n) + 2(\frac{n}{2}) = 2(45) \rightarrow 4n + n = 90$.
    2.  Sumamos las letras: $5n = 90$.
    3.  Dividimos entre 5: $n = 90 / 5 \rightarrow n = 18$.
*   **Paso 4 (Verificación):** El doble de 18 es 36. La mitad de 18 es 9. $36 + 9 = 45$. ¡Correcto!
*   **Respuesta:** El número es 18.

### Ejemplo B: Números Consecutivos
**Problema:** La suma de dos números consecutivos es 451. Hallar los números.

*   **Paso 1 (Variable):** $n$ es el menor, $n + 1$ es el mayor.
*   **Paso 2 (Ecuación):** $n + (n + 1) = 451$
*   **Paso 3 (Resolución):**
    1.  Agrupamos: $n + n = 451 - 1$.
    2.  Operamos: $2n = 450$.
    3.  Despejamos: $n = 450 / 2 \rightarrow n = 225$.
*   **Paso 4 (Verificación):** Si el menor es 225, el consecutivo es 226. $225 + 226 = 451$. ¡Correcto!
*   **Respuesta:** Los números son 225 y 226.

### Ejemplo C: Problema de Edades
**Problema:** Un padre tiene 37 años y su hijo 9. ¿Dentro de cuántos años la edad del padre será el triple de la del hijo?

*   **Paso 1 (Variable):** $a$ es el número de años que deben pasar.
*   **Paso 2 (Ecuación):** $(37 + a) = 3 \times (9 + a)$
*   **Paso 3 (Resolución):**
    1.  Multiplicamos el paréntesis: $37 + a = 27 + 3a$.
    2.  Pasamos las letras a un lado: $37 - 27 = 3a - a$.
    3.  Operamos: $10 = 2a$.
    4.  Despejamos: $10 / 2 = a \rightarrow a = 5$.
*   **Paso 4 (Verificación):** En 5 años el padre tendrá 42 y el hijo 14. 42 es el triple de 14. ¡Correcto!
*   **Respuesta:** Sucederá dentro de 5 años.

## 4. Cuestionario de Repaso (Preguntas Cortas)

Resuelve los siguientes ejercicios aplicando los pasos aprendidos:

1.  **Diferencia y fracciones:** La diferencia entre el triple de un número y su quinta parte es 70. ¿Cuál es el número?
2.  **Igualdad de expresiones:** El triple de un número disminuido en 8 equivale al mismo número aumentado en 24. Hallar el número.
3.  **Tres consecutivos:** La suma de tres números consecutivos es 81. ¿Cuáles son los números?
4.  **Antecesor y sucesor:** La diferencia entre el quíntuple del antecesor de un número y el triple de su sucesor es 24. ¿De qué número se trata?
5.  **Suma de edades:** Las edades de dos hermanos suman 34 años. Si uno es 4 años mayor que el otro, ¿qué edad tiene cada uno?

1.  **Análisis Comparativo de Métodos:** Explica las ventajas y desventajas de resolver un problema matemático mediante el razonamiento mental (lógica) frente al uso de ecuaciones algebraicas. ¿En qué situaciones es indispensable usar ecuaciones?
2.  **La Importancia de la Verificación:** Describe por qué el proceso de solución de una ecuación no termina cuando se halla el valor de la variable ($n$). ¿Qué papel juega la verificación en la seguridad del estudiante y en la resolución de problemas complejos?
3.  **El Lenguaje como Herramienta:** Reflexiona sobre cómo el lenguaje cotidiano puede ser ambiguo en comparación con el lenguaje algebraico. Utiliza el ejemplo de "la diferencia" o "el sucesor" para explicar cómo la precisión matemática ayuda a evitar confusiones.

## 6. Glosario de Términos Importantes

*   **Ecuación:** Una igualdad matemática entre dos expresiones que contiene una o más incógnitas.
*   **Ecuación Equivalente:** Ecuaciones que tienen la misma solución. Se obtienen al realizar la misma operación (como multiplicar o dividir) en ambos lados de la igualdad.
*   **Incógnita (Variable):** La cantidad desconocida que se quiere encontrar en un problema, representada generalmente por una letra.
*   **Diferencia:** El resultado de una operación de resta.
*   **Término:** Cada una de las partes de una expresión algebraica separadas por los signos de suma o resta.
*   **Simplificar:** Proceso de reducir una expresión u operación a su forma más sencilla o corta (por ejemplo, eliminando denominadores o sumando términos semejantes).
*   **Equivaler:** Palabra que en un problema matemático indica que se debe colocar el signo de igualdad ($=$).
