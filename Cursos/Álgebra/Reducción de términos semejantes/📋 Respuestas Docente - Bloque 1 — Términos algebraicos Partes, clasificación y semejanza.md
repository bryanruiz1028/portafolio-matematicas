## 1. Clave de Respuestas Detallada

*   **Q1: b)** Expresión que tiene solamente un término. Según el autor, el prefijo "mono" indica unidad.
*   **Q2: c)** $-1$. Justificación: El Profe Alex enseña que cuando no hay número visible, el coeficiente es 1, y el signo negativo siempre debe acompañar al coeficiente numérico.
*   **Q3: c)** Trinomio. Posee tres términos claramente separados por signos.
*   **Q4: Verdadero.** Es la regla técnica fundamental para el conteo de términos en polinomios.
*   **Q5: Falso.** Se enfatiza que en matemáticas la capitalización (mayúscula vs. minúscula) rompe la semejanza; son variables distintas.
*   **Q6:** (6a) b-más / (6b) d-menos. (El orden es intercambiable).
*   **Q7:** (7a) c-parte literal / (7b) e-letras / (7c) a-exponentes. 
*   **Q8 (Relacionar):** 

| Expresión Algebraica | R | Clasificación |
| :--- | :---: | :--- |
| 1. $5x$ | **b** | Monomio |
| 2. $2a + 3b$ | **a** | Binomio |
| 3. $x + y + z$ | **d** | Trinomio |
| 4. $a + b + c + d$ | **c** | Polinomio |

*   **Q9 (Identificación):** Signo: Negativo ($-$), Coeficiente: $-3$, Variable: $x$, Exponente: 2, Parte Literal: $x^2$. (Nota: La parte literal debe incluir la variable y el exponente).
*   **Q10 (Aplicación):** Expresión: $5x + 3x$. Procedimiento: Al ser términos semejantes (misma variable $x$ con mismo exponente 1), se suman los coeficientes. Resultado: $8x$.
