> [!info] 🌍 Relevancia y aplicaciones
> Traducir el lenguaje cotidiano a ecuaciones matemáticas es la habilidad fundamental para modelar la realidad y resolver problemas de optimización. Mientras que la lógica simple funciona en casos elementales, el álgebra es nuestra "herramienta de poder" para gestionar situaciones donde la complejidad supera el razonamiento intuitivo.
> - **💵 Aplicación con $USD:** Cálculo exacto de presupuestos para la compra de tecnología o entradas a eventos masivos, ajustando costos fijos y variables.
> - **🏗️ Aplicación práctica:** Determinación de dimensiones críticas en construcción, diseño técnico y geometría, asegurando que las piezas encajen perfectamente.
> - **📊 Situación cotidiana:** Organización logística de grupos y distribución equitativa de recursos en eventos sociales o entornos corporativos.

---

> [!note] 📌 ¿Qué es Solución de problemas con Ecuaciones de Primer Grado?
> Es el proceso intelectual de traducir enunciados verbales al lenguaje simbólico (números y letras) para hallar una incógnita. Consiste en construir un puente entre la narrativa del problema y la estructura lógica de la igualdad matemática.

> [!warning] ⚠️ Error común
> **❌ Incorrecto:** Dar por finalizado el ejercicio al despejar la incógnita sin evaluar el contexto.
> **✅ Correcto:** Realizar una verificación lógica. Si el resultado es una fracción de persona (ej. $10.5$ invitados) o un valor negativo donde no corresponde, el planteamiento es erróneo. Los resultados deben tener sentido en el mundo real.

> [!tip] 💡 Truco para recordarlo
> Implementa siempre la metodología **"L.E.R.V."**:
> 1. **L**eer: Comprender el enunciado profundamente.
> 2. **E**scribir: Traducir los datos al lenguaje algebraico.
> 3. **R**esolver: Ejecutar las operaciones para despejar la variable.
> 4. **V**erificar: Comprobar si el valor satisface las condiciones originales.

---

### Procedimiento paso a paso

Como especialistas, seguimos una estructura rigurosa para minimizar errores de planteamiento. Un consejo vital: **asigna siempre la incógnita ($x$) al objeto o valor más pequeño o barato**; esto evita trabajar con fracciones o restas innecesarias.

```text
PASO 1 → Identificar la incógnita y darle nombre (usar letras mnemotécnicas: 'h' para hombres, 'c' para camisas). 
         REGLA DE ORO: Asignar la variable al valor menor.
PASO 2 → Traducir el enunciado al lenguaje algebraico para construir la ecuación.
PASO 3 → Resolver la ecuación (organizar términos: variables a un lado, constantes al otro).
PASO 4 → Verificar el resultado en el contexto original del problema.
```

### Ejemplos prácticos

```ad-example
title: Ejemplo 1 — Razones y proporciones
**Enunciado:** Dos números están en razón de $7$ a $9$. Si cada uno aumenta en $4$ unidades, la razón cambia a $4$ a $5$.
- **Didáctica:** Usamos una constante multiplicadora $n$ porque toda razón es una fracción simplificada. Multiplicar por $n$ nos permite trabajar con los valores reales.
- **Planteamiento:** 
	- **Antecedente** (número superior): $7n$
	- **Consecuente** (número inferior): $9n$
	- Ecuación: $\frac{7n + 4}{9n + 4} = \frac{4}{5}$
- **Resolución:** Multiplicamos en cruz: $5(7n + 4) = 4(9n + 4) \rightarrow 35n + 20 = 36n + 16$.
- **Resultado:** $n = 4$. Los números iniciales son $7(4) = 28$ y $9(4) = 36$.
```

```ad-example
title: Ejemplo 2 — Composición de grupos
**Enunciado:** En una fiesta hay el doble de mujeres que de hombres, y hay $4$ niños más que mujeres. En total hay $54$ personas.
- **Planteamiento:** 
	- Hombres (valor menor): $h$
	- Mujeres: $2h$
	- Niños: $2h + 4$
	- Ecuación: $h + 2h + (2h + 4) = 54$
- **Resolución:** $5h + 4 = 54 \rightarrow 5h = 50 \rightarrow h = 10$.
- **Resultado:** Hay $10$ hombres, $20$ mujeres y $24$ niños. Suma total: $10+20+24 = 54$.
```

```ad-example
title: Ejemplo 3 — Ángulos de un triángulo
**Enunciado:** En un triángulo, el ángulo $A$ mide el doble que el $B$, y el $C$ mide $20°$ menos que el $B$.
- **Didáctica:** Recordamos la propiedad fundamental: $\angle A + \angle B + \angle C = 180°$.
- **Planteamiento:** 
	- $B = B$
	- $A = 2B$
	- $C = B - 20$
	- Ecuación: $2B + B + (B - 20) = 180$
- **Resolución:** $4B - 20 = 180 \rightarrow 4B = 200 \rightarrow B = 50$.
- **Resultado:** $\angle A = 100°$, $\angle B = 50°$ y $\angle C = 30°$.
```

```ad-example
title: Ejemplo 4 — Entradas al parque con $USD
**Enunciado:** Se compran entradas para $5$ niños y $4$ adultos por un total de $820$ USD. La entrada de adulto cuesta $10$ USD más que el doble de la de un niño.
- **Planteamiento:** 
	- Entrada niño (más barata): $x$
	- Entrada adulto (doble más 10): $2x + 10$
	- Ecuación: $5x + 4(2x + 10) = 820$
- **Resolución:** $5x + 8x + 40 = 820 \rightarrow 13x = 780 \rightarrow x = 60$.
- **Resultado:** La entrada de niño cuesta $60$ USD y la de adulto $130$ USD.
```

### Ejercicios para el estudiante

```ad-abstract
title: 🟢 Fácil — Razones numéricas
Dos números están en razón $4:5$. Si el primero aumenta en $6$ y el segundo disminuye en $5$, la nueva razón es $6:5$. Halle los números iniciales.
```

```ad-abstract
title: 🟡 Medio — Compras y cantidades
Se compraron $6$ pantalones y $4$ camisas por un total de $480$ USD. Si cada pantalón costó $5$ USD más que cada camisa, ¿cuál es el precio unitario de cada prenda?
```

```ad-abstract
title: 🔴 Avanzado — Geometría e Isósceles
El perímetro de un triángulo isósceles es de $70$ cm. Si la base mide $10$ cm más que uno de sus lados iguales, ¿cuánto mide cada lado?
```

```ad-success
title: ✅ Respuestas (Guía docente)
- **🟢 Fácil:** Los números son $24$ y $30$.
	- *Planteamiento:* $\frac{4x + 6}{5x - 5} = \frac{6}{5}$
- **🟡 Medio:** Camisa $45$ USD, Pantalón $50$ USD.
	- *Planteamiento:* $x = \text{camisa}$; $6(x + 5) + 4x = 480$
- **🔴 Avanzado:** Lados iguales $20$ cm, Base $30$ cm.
	- *Planteamiento:* $x + x + (x + 10) = 70$
```

### Autoevaluación

```ad-question
title: 🧪 Pregunta 1
¿Cuál es la propiedad geométrica que permite plantear ecuaciones sobre los ángulos internos de un triángulo?
- **Respuesta:** ✅ La suma de los tres ángulos internos siempre debe ser igual a $180°$.
```

```ad-question
title: 🧪 Pregunta 2
Si $x$ representa el costo de un artículo, ¿cómo se traduce "el triple de un número aumentado en 15"?
- **Respuesta:** ✅ $3x + 15$. El término "triple" implica multiplicación por $3$ y "aumentado" implica suma.
```

```ad-question
title: 🧪 Pregunta 3
Un padre tiene el triple de la edad de su hijo. Si la suma de sus edades es $60$ años, ¿qué edad tiene el hijo?
- **Respuesta:** ✅ $15$ años. Si el hijo es $x$ y el padre es $3x$, entonces $x + 3x = 60 \rightarrow 4x = 60 \rightarrow x = 15$.
```
