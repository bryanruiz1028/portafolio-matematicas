### 2. RESOLUCIÓN DETALLADA (Q9 Y Q10)

**Q9: Números Consecutivos**
1.  **Definición:** Sea $n$ el número menor y $n+1$ el consecutivo.
2.  **Ecuación:** $n + (n + 1) = 451$.
3.  **Despeje:** $2n + 1 = 451 \rightarrow 2n = 450 \rightarrow n = 225$.
4.  **Verificación:** $225 + 226 = 451$.

**Q10: Parque de Diversiones**
1.  **Asignación de Variables:** Profe Alex recomienda asignar la variable al valor menor. Sea $n$ = precio entrada niño. Entonces, el precio adulto es $2n + 10$.
2.  **Ecuación:** $5n + 4(2n + 10) = 820$.
3.  **Despeje:** 
    *   $5n + 8n + 40 = 820$
    *   $13n + 40 = 820 \rightarrow 13n = 780$
    *   $n = 780 / 13 = 60$.
4.  **Resultados:** Niño = 60 USD; Adulto = $2(60) + 10 = 130$ USD.
