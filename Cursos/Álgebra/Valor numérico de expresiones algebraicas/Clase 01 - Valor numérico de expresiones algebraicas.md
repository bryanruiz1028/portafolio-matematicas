## 1. Relevancia y Aplicaciones

> [!info] 🌍 ¿Por qué es importante esta clase?
> El **valor numérico** es lo que nos permite aterrizar el álgebra a situaciones de la vida real. Sin él, las fórmulas serían solo letras sin sentido. Al reemplazar las variables por cantidades conocidas, podemos tomar decisiones informadas:
> - 💵 **Finanzas:** Calcular impuestos o presupuestos en $USD usando fórmulas de ahorro.
> - 🏗️ **Construcción:** Determinar áreas de terreno o materiales necesarios en una obra.
> - 📊 **Predicciones:** Aplicar fórmulas para proyectar cuánto dinero habrás ahorrado en el futuro.

---

## 2. Fundamentación Teórica

> [!note] 📌 ¿Qué es el Valor Numérico?
> Para un estudiante que está empezando, es muy simple: consiste en **reemplazar las letras por números** que nos dan en el ejercicio y luego **resolver las cuentas** (las operaciones) siguiendo el orden correcto.

> [!warning] ⚠️ Error común: Las letras son nombres propios
> En matemáticas, las minúsculas y las mayúsculas son **nombres diferentes**. Si el ejercicio dice que a = 2, esto solo aplica para la "a" minúscula. Si ves una "A" mayúscula, no puedes usar ese 2 porque para el álgebra son entidades distintas. ¡No las confundas!

> [!warning] ⚠️ El peligro de los Signos y Exponentes
> No es lo mismo tener (-5)^2 que -5^2. 
> - En **(-5)^2**, el exponente afecta a todo el paréntesis: (-5) * (-5) = 25.
> - En **-5^2**, el exponente solo afecta al 5, y el negativo se queda esperando afuera: -(5 * 5) = -25.
> **¡Usa siempre paréntesis al sustituir!**

> [!tip] 💡 Truco para recordarlo
> Imagina que las letras son **"cajas vacías"**. El proceso de hallar el valor numérico es simplemente quitar la letra, poner un paréntesis (la caja) y meter el número dentro para luego operar.

## 3. Metodología Procedimental

Para no equivocarte nunca, acostúmbrate a trabajar **hacia abajo**. Esto te permite ver cada paso y detectar errores rápidamente:

```text
PASO 1 → Identificar los valores asignados a cada variable (letra).
PASO 2 → Reemplazar cada letra por su valor entre paréntesis.
         (Es obligatorio si el número es negativo o fracción).
PASO 3 → Respetar la jerarquía de operaciones: 
         Primero Potencias, luego Multiplicaciones, al final Sumas y Restas.
PASO 4 → Operar término a término, resolviendo de forma vertical.
```

## 4. Casos de Estudio (Ejemplos Resueltos)

> [!example] Ejemplo 1: Operación básica (Suma)
> **Expresión:** a + b + c
> **Valores:** a = 2, b = 3, c = 5
> 
> **Solución vertical:**
> a + b + c
> = 2 + 3 + 5
> = 10

> [!example] Ejemplo 2: Coeficientes y Multiplicación
> **Expresión:** 3a + 4b
> **Valores:** a = 2, b = 3
> 
> **Solución vertical:**
> 3(2) + 4(3)  *(Nota: El paréntesis indica multiplicación).*
> = 6 + 12
> = 18
> *⚠️ Error a evitar: No sumes 2 + 4 antes de multiplicar.*

> [!example] Ejemplo 3: Fracciones y Simplificación
> **Expresión:** x + 2y
> **Valores:** x = 1/2, y = 3/4
> 
> **Solución vertical:**
> 1/2 + 2(3/4)
> = 1/2 + 6/4  *(Multiplicamos 2 * 3 y mantenemos el denominador).*
> = 1/2 + 3/2  *(Simplificamos 6/4 sacando mitad).*
> = 4/2
> = 2          *(Como son fracciones homogéneas, sumamos directo).*

> [!example] Ejemplo 4: Aplicación Real ($USD)
> **Problema:** El costo de un servicio es 2m + 3n, donde "m" es el precio por hora y "n" es el cargo de envío. Si m = 1/2 USD y n = 2/3 USD, ¿cuál es el costo total?
> 
> **Solución vertical:**
> 2(1/2) + 3(2/3)
> = 2/2 + 6/3
> = 1 + 2
> = 3 USD

## 5. Práctica y Evaluación

> [!abstract] 🟢 Nivel Fácil
> Calcula el valor de: a - b + c
> Si: a = 2, b = 3, c = 5.

> [!abstract] 🟡 Nivel Medio
> Calcula el valor de: 5a - 2b
> Si: a = 2, b = 3.

> [!abstract] 🔴 Nivel Avanzado
> Calcula el valor de: x^2 + 2z
> Si: x = 1/2, z = -2.

> [!success] 🔑 Clave de Respuestas
> 1. **Fácil:** 4
> 2. **Medio:** 4
> 3. **Avanzado:** -15/4 (que es -3.75).

> [!question] 📝 Mini-prueba: Conceptual
> **Si en un ejercicio tienes "a = 5" y la expresión es "2A + 3", ¿qué debes hacer?**
> A) Reemplazar la A por 5.
> B) Dejar la A como está, porque "a" y "A" son nombres distintos.
> C) Sumar 5 + 2 y luego multiplicar.
> **Respuesta:** B. En matemáticas son variables diferentes.

> [!question] 📝 Mini-prueba: Procedimental
> **En la operación 4 + 3(2^2), ¿qué se resuelve primero?**
> A) La suma 4 + 3.
> B) La potencia 2^2.
> C) La multiplicación 3 * 2.
> **Respuesta:** B. Según la jerarquía, las potencias van antes que multiplicaciones y sumas.

# Guía de Estudio: Valor Numérico de Expresiones Algebraicas (Básica Superior)

Esta guía resume los puntos vitales para dominar el álgebra desde cero, siguiendo el método del Profe Alex.

### 1. Resumen de Conceptos Clave
- **Variable:** Una letra que guarda el lugar de un número que conoceremos después.
- **Sustituir:** Quitar la letra y poner el número en su lugar (¡siempre entre paréntesis!).
- **Trabajo Vertical:** Resolver paso a paso hacia abajo para no perder el hilo de la cuenta.

### 2. El "Acordeón" Matemático (Fórmulas)

**Ley de Signos para Multiplicación y División**
- (+) por (+) = +
- (-) por (-) = + (Signos iguales dan positivo)
- (+) por (-) = -
- (-) por (+) = - (Signos diferentes dan negativo)

**Regla de Oro para Suma y Resta**
- **Signos iguales:** Se suman los números y se deja el mismo signo.
- **Signos diferentes:** Se restan (el mayor menos el menor) y se deja el signo del que tenga el mayor valor absoluto (el número más "grande" visualmente).

**Jerarquía de Operaciones**
1. Paréntesis (lo de adentro primero).
2. Potencias y Raíces.
3. Multiplicaciones y Divisiones.
4. Sumas y Restas.

### 3. Refuerzo Visual: Fracciones (El método de la X)
Cuando tengas fracciones con distinto denominador (heterogéneas), usa el método de la **"Carita Feliz"**:
Para la operación: `1/2 + 2/3`
1. Multiplica en cruz: `(1 * 3) = 3` y `(2 * 2) = 4`.
2. Suma esos resultados: `3 + 4 = 7`.
3. Multiplica los de abajo (denominadores): `2 * 3 = 6`.
4. Resultado final: `7/6`.

### 4. Desafío Final
Resuelve estos ejercicios usando: **a = 2, b = 3, c = 5, x = 1/2**

1. b^2 + a
2. 2c - 3b
3. (1/2) * a + b
4. (a + b) / c

Recuerda lo que siempre decimos: la matemática no se aprende solo mirando, se aprende haciendo. Si te equivocas en un signo, no te desanimes; es el error más común del mundo. Revisa tu procedimiento **hacia abajo** y verás que cada vez te volverás más rápido y preciso. ¡A practicar!
