### SECCIÓN IV — Relacionar columnas (Q8)
*Instrucción: Vincule la Propiedad con su Definición o Expresión Simbólica escribiendo la letra correspondiente en la columna "R".*

| Propiedad | Definición / Ejemplo | R |
| :--- | :--- | :--- |
| 1. Potencia de una potencia | A. Se traslada la base al denominador para cambiar el signo: $a^{-n} = 1/a^n$. | |
| 2. Exponente negativo (fracción) | B. Se mantiene la base y se multiplican los exponentes: $(a^m)^n = a^{m \cdot n}$. | |
| 3. Exponente negativo (entero) | C. Se invierte la posición de los términos: $(a/b)^{-n} = (b/a)^n$. | |
| 4. Exponente cero | D. El resultado es siempre 1, siempre que la base sea distinta de cero. | |
