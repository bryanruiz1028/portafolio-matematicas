# 📖 Guía de estudio — Clase 02: Razones trigonométricas de un ángulo

> [!info] 📌 Conceptos clave
> - **Definición:** Una razón trigonométrica es la relación (división) entre las longitudes de dos lados de un triángulo rectángulo respecto a uno de sus ángulos agudos.
> - **El ángulo de referencia:** ¡Este es el paso más importante! Dependiendo de qué ángulo elijas, los nombres de los catetos cambian. El **cateto opuesto** siempre está enfrente del ángulo, mientras que el **cateto adyacente** es el que forma parte del ángulo.
> - **Conocimiento total de los lados:** No puedes calcular las razones si te falta un lado. Si el problema no te da los tres, usa el **Teorema de Pitágoras** para encontrar el valor faltante.
> - **Adiós a las unidades:** Al dividir longitudes (como $m$ o $cm$), las unidades se cancelan. El resultado es un valor numérico puro que representa la proporción.

---

## Fórmulas y definiciones

| Término | Definición / Fórmula |
| :--- | :--- |
| **Seno** | $\frac{CO}{H}$ |
| **Coseno** | $\frac{CA}{H}$ |
| **Tangente** | $\frac{CO}{CA}$ |
| **Cotangente** | $\frac{CA}{CO}$ |
| **Secante** | $\frac{H}{CA}$ |
| **Cosecante** | $\frac{H}{CO}$ |

---

## Ejemplos resueltos adicionales

```ad-example
title: Ejemplo A — Identificación y cálculo básico
**Problema:** En un triángulo con medidas de $3m, 4m$ y $5m$, calcula las razones principales para el ángulo $\theta$.

**Paso a paso:**
1. **Identificar los lados:** Mirando el ángulo $\theta$, el lado de $3m$ está enfrente, así que es el **Cateto Opuesto ($CO$)**. El lado de $4m$ está junto al ángulo, es el **Cateto Adyacente ($CA$)**. El lado más largo ($5m$) es la **Hipotenusa ($H$)**.
2. **Seno:** $\frac{CO}{H} = \frac{3}{5} = 0.6$
3. **Coseno:** $\frac{CA}{H} = \frac{4}{5} = 0.8$
4. **Tangente:** $\frac{CO}{CA} = \frac{3}{4} = 0.75$
```

```ad-example
title: Ejemplo B — Hallar la hipotenusa con Pitágoras
**Escenario:** Se diseña una rampa de madera. El soporte vertical mide $4cm$ y la base horizontal mide $3cm$. Queremos saber cuánto mide la rampa (hipotenusa) y cuál es su costo total si la madera especial cuesta $12.00$ USD por centímetro lineal.

**Paso a paso:**
1. **Identificar lados (Ángulo $\alpha$ en la base):** El soporte vertical ($4cm$) es el **$CO$**. La base horizontal ($3cm$) es el **$CA$**. La rampa es la **$H$** (desconocida).
2. **Calcular la hipotenusa con Pitágoras:**
   $H^2 = CO^2 + CA^2$
   $H^2 = 4^2 + 3^2$
   $H^2 = 16 + 9 = 25$
   $H = \sqrt{25} = 5cm$
3. **Calcular el Seno:** $\frac{CO}{H} = \frac{4}{5} = 0.8$
4. **Cálculo de costo ($USD):** Si la hipotenusa mide $5cm$ y cada cm cuesta $12.00$ USD:
   $5 \times 12.00 = 60.00$
   **Resultado:** El costo total de la rampa es de **$60.00 USD**.
```

## Ejercicios de repaso

```ad-abstract
title: 🟢 Fácil
En un triángulo rectángulo, el ángulo de referencia tiene un cateto opuesto de $6$, un cateto adyacente de $8$ y una hipotenusa de $10$. Calcula:
1. El **Seno** en fracción y decimal.
2. El **Coseno** en fracción y decimal.
```

```ad-abstract
title: 🟡 Medio
Calcula las 6 razones trigonométricas en forma decimal para el ángulo $\alpha$ con los siguientes datos exactos:
- Cateto opuesto ($CO$) = $10$
- Cateto adyacente ($CA$) = $6$
- Hipotenusa ($H$) = $11.6$
```

```ad-abstract
title: 🔴 Avanzado — Aplicación con $USD
Una pieza de ingeniería tiene catetos de $4$ y $6$ unidades. 
1. Verifica la hipotenusa usando Pitágoras (debe dar aprox. $7.2$).
2. Determina la **Secante** del ángulo $\alpha$ (considerando que el cateto adyacente es $4$).
3. Si el material para la hipotenusa tiene un precio de **$15.50 USD** por unidad de medida, ¿cuál es el costo total de este lado del proyecto? (Usa $7.2$ para el cálculo).
```

---

> [!tip] 💡 Consejo de estudio
> Para no memorizar 6 fórmulas, apréndete solo las 3 primeras (**Seno, Coseno, Tangente**). Las otras 3 son simplemente sus **parejas inversas** (la fracción se voltea):
> - El **Seno** es pareja de la **Cosecante**.
> - El **Coseno** es pareja de la **Secante**.
> - La **Tangente** es pareja de la **Cotangente**.
> 
> *¡Si el Seno es $3/5$, la Cosecante será automáticamente $5/3$!*
