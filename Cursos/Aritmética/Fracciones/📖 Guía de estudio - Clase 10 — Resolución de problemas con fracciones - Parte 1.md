### 📚 Guía de Estudio: Resolución de Problemas con Fracciones

#### 1. Cuadro Resumen de Operaciones
| Objetivo (Meta) | Situación Clave | Operación |
| :--- | :--- | :--- |
| **Sumar partes** | Acumular varias porciones. | Suma (`Carita Feliz`) |
| **Hallar una parte** | Tienes el total y buscas un trozo. | Multiplicación (`Fracción * Total`) |
| **Hallar el total** | Tienes el valor de un trozo y buscas el todo. | División (`Valor / Fracción`) |

#### 2. Fórmulas y Métodos
- **Método Carita Feliz:** Úsalo para sumar fracciones de distinto denominador. `(a/b) + (c/d) = (ad + bc) / bd`.
- **Simplificación:** ¡No la olvides! Divide numerador y denominador por el mismo número hasta que no se pueda más.
- **Lo que falta para la unidad:** Si tienes `x/y`, lo que falta es `(y-x)/y`.

#### 3. Checklist para el Éxito
1. [ ] **Leer dos veces:** La primera para entender, la segunda para extraer datos.
2. [ ] **Dibujar la tabla:** Separa dinero/personas de las fracciones.
3. [ ] **Identificar el "1":** ¿Qué es el total en este problema?
4. [ ] **Elegir operación:** ¿Busco un trozo (multiplico) o el todo (divido)?

#### 4. Ejemplo de Repaso (Combinado)
Si tienes 120 USD y gastas `1/2` en comida y `1/3` en transporte:
- Gasto total: `1/2 + 1/3 = 5/6`.
- Te queda: `1 - 5/6 = 1/6`.
- Dinero final: `120 * 1/6 =` **20 USD**.

#### 5. 5 Ejercicios de Repaso Rápido
1.  ¿Cuánto es `2/5` de 500? (R: **200**)
2.  Si una obra lleva `4/9`, ¿qué fracción falta? (R: **5/9**)
3.  Sumar `2/7 + 1/3`. (R: **13/21**)
4.  Si 40 USD son `1/4` del total, ¿cuánto es el total? (R: **160 USD**)
5.  Simplifica `15/45`. (R: **1/3**)
