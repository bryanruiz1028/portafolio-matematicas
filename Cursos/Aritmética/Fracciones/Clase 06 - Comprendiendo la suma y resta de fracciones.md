¡Qué tal amigos! Espero que estén muy bien. Bienvenidos al curso de fracciones, donde hoy vamos a entender no solo cómo se operan, sino qué es lo que realmente sucede cuando sumamos, restamos, multiplicamos o dividimos estas cantidades.

> [!info] 🌍 Relevancia y aplicaciones
> Dominar las cuatro operaciones básicas con fracciones es la herramienta fundamental para resolver problemas lógicos y financieros con precisión. Entender estos procesos permite que las matemáticas dejen de ser reglas de memoria y se conviertan en soluciones prácticas para el día a día.
> - 💵 **Aplicación USD:** Calcular rápidamente el vuelto o cambio al pagar una deuda de $\frac{1}{4}$ de un billete de $\$100$ usando el billete entero.
> - 🏗️ **Práctica:** Medir materiales de construcción con exactitud cuando las dimensiones vienen expresadas en pulgadas o fracciones de unidad.
> - 📊 **Cotidiana:** Repartir recursos de forma justa (como una pizza o pastel) o gestionar el tiempo calculando fracciones de hora en minutos.

> [!note] 📌 ¿Qué es Comprendiendo la suma y resta de fracciones?
> Imagina que tienes una chocolatina. Sumar o restar fracciones es simplemente **juntar o quitar partes de un mismo tamaño**. Si las partes tienen tamaños distintos, no podemos contarlas directamente; primero debemos "reajustarlas" (amplificarlas) para que todas las piezas sean iguales antes de operar con ellas.

> [!warning] ⚠️ Error común
> ❌ **Incorrecto:** Sumar los denominadores directamente. Ejemplo: $\frac{1}{2} + \frac{1}{2} = \frac{2}{4}$. (¡Esto es imposible, pues dos mitades forman un entero!).
> ✅ **Correcto:** Mantener el denominador si es igual y solo operar los numeradores. Ejemplo: $\frac{1}{2} + \frac{1}{2} = \frac{1+1}{2} = \frac{2}{2} = 1$.

> [!tip] 💡 Truco para recordarlo
> Cuando las fracciones tienen el mismo denominador, decimos que **"las fracciones se vuelven amigas"** (son homogéneas) y podemos sumarlas o restarlas de inmediato. Si son diferentes, usamos la técnica de la **"carita feliz"** o la **"amplificación"** para que se vuelvan amigas.

### PROCEDIMIENTO PASO A PASO

```text
SUMA Y RESTA:
PASO 1 → Identificar si los denominadores son iguales (homogéneas) o diferentes (heterogéneas).
PASO 2 → Si son diferentes, hallar el Mínimo Común Múltiplo (MCM) para amplificar las fracciones.
PASO 3 → Operar solo los numeradores manteniendo el denominador común.
PASO 4 → Simplificar el resultado a su mínima expresión.

MULTIPLICACIÓN:
PASO 1 → Multiplicar "de frente": numerador por numerador y denominador por denominador.

DIVISIÓN:
PASO 1 → Multiplicar los "Extremos" para obtener el nuevo numerador.
PASO 2 → Multiplicar los "Medios" para obtener el nuevo denominador.
```

### EJEMPLOS DE APLICACIÓN

```ad-example
title: **Ejemplo 1: Suma de fracciones homogéneas**
Operación: $\frac{3}{7} + \frac{2}{7}$
Como ya son "amigas" (denominador 7), solo sumamos arriba:
$\frac{3 + 2}{7} = \frac{5}{7}$
*Nota: Esta fracción no se puede simplificar más.*
```

```ad-example
title: **Ejemplo 2: Resta con denominadores distintos**
Operación: $\frac{5}{4} - \frac{1}{2}$
Para que sean amigas, amplificamos $\frac{1}{2}$ multiplicando por 2: $\frac{1 \times 2}{2 \times 2} = \frac{2}{4}$.
Ahora restamos: $\frac{5}{4} - \frac{2}{4} = \frac{3}{4}$
*Resultado: $\frac{3}{4}$*
```

```ad-example
title: **Ejemplo 3: Operación combinada (MCM)**
Operación: $\frac{1}{4} + \frac{8}{3} - \frac{5}{6}$
1. El MCM de 4, 3 y 6 es **12**.
2. Convertimos todas a doceavos:
   - $\frac{1 \times 3}{4 \times 3} = \frac{3}{12}$
   - $\frac{8 \times 4}{3 \times 4} = \frac{32}{12}$
   - $\frac{5 \times 2}{6 \times 2} = \frac{10}{12}$
3. Operamos: $\frac{3 + 32 - 10}{12} = \frac{25}{12}$
```

```ad-example
title: **Ejemplo 4: Ahorros en USD**
Si tienes ahorrado $\frac{1}{2}$ de dólar y recibes $\frac{1}{4}$ de dólar adicional:
$\frac{1}{2} + \frac{1}{4} = \frac{2}{4} + \frac{1}{4} = \frac{3}{4}$
*Resultado: Tienes $\frac{3}{4}$ de dólar, que son 75 centavos de dólar.*
```

### AUTOEVALUACIÓN

```ad-question
title: **Pregunta 1 (Conceptual)**
¿Qué sucede con el denominador cuando sumamos dos fracciones que ya son "amigas" (homogéneas)?
**Respuesta:** El denominador se mantiene exactamente igual; solo se operan los numeradores.
```

```ad-question
title: **Pregunta 2 (Procedimental)**
Calcula rápidamente aplicando la multiplicación lineal: $\frac{2}{3} \times \frac{3}{5}$
**Respuesta:** $\frac{6}{15}$. Si simplificamos dividiendo por 3, obtenemos $\frac{2}{5}$.
```

```ad-question
title: **Pregunta 3 (Aplicación $USD)**
Si tienes 5 dólares y gastas $\frac{3}{8}$ de un dólar, ¿cuánto dinero te queda en fracción?
**Respuesta:** $\frac{37}{8}$ de dólar.
*Explicación mental:* $5 \times 8 = 40$. Entonces $\frac{40}{8} - \frac{3}{8} = \frac{37}{8}$.
```
