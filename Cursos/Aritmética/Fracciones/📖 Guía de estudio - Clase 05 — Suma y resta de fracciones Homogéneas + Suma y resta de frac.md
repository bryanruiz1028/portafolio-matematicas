# 📖 Guía de estudio — Clase 05: Suma y resta de fracciones homogéneas

¡Qué tal, amigas y amigos! Espero que estén muy bien. En esta guía vamos a aprender a operar con fracciones homogéneas. Como siempre les digo, no me interesa que solo aprendan pasos de memoria, sino que entiendan el **porqué** de las cosas para que este conocimiento les sirva para toda la vida.

> [!info] 📌 Conceptos clave
> * **¿Qué son?**: Las fracciones homogéneas son aquellas que tienen el **mismo denominador**. Esto significa que todas las unidades están divididas en partes del mismo tamaño.
> * **Lógica de la operación**: Para sumar o restar, simplemente operamos los numeradores y **mantenemos el mismo denominador**. ¿Por qué? Porque si estamos contando "tercios" o "cuartos", el resultado lógicamente seguirá siendo en esa misma medida de porciones.
> * **Simplificación**: Una vez obtenido el resultado, buscamos la fracción más simple. Es una señal de un trabajo bien hecho y nos da la respuesta "más correcta".
> * **Escalabilidad**: Este método es universal. Funciona exactamente igual si tienes 2, 3, 4 o muchísimas más fracciones, siempre que compartan el mismo denominador.

## 2. Fórmulas y definiciones importantes

| Término | Definición / Fórmula |
| :--- | :--- |
| **Fracciones Homogéneas** | Son fracciones que poseen el mismo denominador (el número de abajo es igual). |
| **Operación de Suma/Resta** | $(a/c) \pm (b/c) = (a \pm b) / c$ (donde $c \neq 0$). |
| **Simplificación** | Dividir el numerador y el denominador por un mismo número (como mitad, tercera o quinta) para obtener la fracción irreducible o un entero. |
| **Amplificación** | Multiplicar el numerador y el denominador por un mismo número para encontrar una fracción **exactamente igual** (equivalente), pero escrita con números más grandes. |

## 3. Ejemplos resueltos adicionales

> [!example] **Ejemplo A — Caso Básico (Operación combinada)**
> **Operación:** $8/12 + 9/12 - 10/12$
> 
> *   **Paso 1 (Identificar):** Todas las fracciones son doceavos. Como estamos contando porciones de tamaño "12", el resultado conservará ese denominador.
> *   **Paso 2 (Operar numeradores):** Realizamos las operaciones de arriba directamente: $8 + 9 - 10$.
>     *   $8 + 9 = 17$
>     *   $17 - 10 = 7$
> *   **Resultado:** $7/12$ ✅. 
> *   *Nota del Profe:* En este caso, $7/12$ no se puede simplificar porque no hay un número que divida exactamente al 7 y al 12 al mismo tiempo.

> [!example] **Ejemplo B — Aplicación real con $USD**
> **Problema:** Si gastas $1/5$ de un dólar en un dulce y luego $2/5$ de dólar en un sobre de láminas, ¿cuánto has gastado en total?
> 
> *   **Paso 1:** Planteamos la suma: $1/5 + 2/5$.
> *   **Paso 2:** Como hablamos de quintos (cinco partes de un dólar), mantenemos el $5$ abajo.
> *   **Paso 3:** Sumamos los numeradores ($1 + 2 = 3$).
> *   **Resultado final:** Has gastado $3/5$ de dólar. ✅

## 5. Consejo de estudio

> [!tip] 💡 Entiende la lógica, no la memoria
> No intentes memorizar pasos mecánicos. Imagina siempre dibujos de pizzas o pasteles divididos. Si tienes porciones del mismo tamaño (denominador igual), solo estás contando cuántas tienes en total; por eso el número de abajo no cambia. 
> 
> **🧠 Reto Mental:** Una vez que domines el papel, intenta hacer las sumas del nivel fácil mentalmente. ¡Es muy sencillo cuando entiendes que solo sumas los números de arriba! Finalmente, recuerda: simplificar es la firma de un buen matemático. ¡A practicar!
