# 📖 Guía de estudio — Clase 02: Resolución de problemas con números enteros

> [!info] 📌 Conceptos clave
> * **Representación espacial y térmica:** Las situaciones que ocurren "sobre" o "arriba" se expresan con números positivos. Lo que está "bajo" o "debajo" (del mar, del suelo o de cero grados) se representa con números negativos.
> * **Sentido de la operación:** Usamos la **suma** para movimientos de ascenso, avance o hacia el futuro (D.C.), y la **resta** para descensos, retrocesos o hacia el pasado (A.C.).
> * **Cálculo de duración:** Para determinar el tiempo transcurrido, se aplica la lógica de **Año final (futuro) menos Año inicial (pasado)**.
> * **Regla de interpretación final:** En la respuesta de un problema, no se escribe el signo negativo. Pedagógicamente, el signo se somete a una **sustitución lingüística** por términos descriptivos como "bajo el nivel del mar", "bajo cero" o "antes de Cristo", facilitando la comprensión del contexto real.

## 2. TABLA DE FÓRMULAS Y DEFINICIONES

| Término | Definición / Fórmula |
| :--- | :--- |
| **Ascender / Subir** | Operación de suma ($+$) |
| **Descender / Bajar** | Operación de resta ($-$) |
| **Bajo cero / Bajo el nivel del mar** | Representación de números negativos ($-$) |
| **Antes de Cristo (A.C.)** | Años representados como números negativos ($-$) |
| **Después de Cristo (D.C.)** | Años representados como números positivos ($+$) |
| **Cálculo de duración / tiempo** | $\text{Año final} - \text{Año inicial}$ |

## 3. EJEMPLOS RESUELTOS ADICIONALES

> [!example] Ejemplo A — El explorador en la cueva
> **Problema:** Un explorador está en una cueva a $50\text{ m}$ bajo el nivel del suelo. Baja $30\text{ m}$, luego sube $65\text{ m}$ y finalmente asciende $12\text{ m}$. ¿Cuál es su posición final?
> 
> **Planteamiento paso a paso:**
> 1.  Posición inicial: $-50$ (bajo el suelo).
> 2.  Baja $30\text{ m}$: $-30$.
> 3.  Sube $65\text{ m}$: $+65$.
> 4.  Asciende $12\text{ m}$: $+12$.
> 
> **Agrupación y resolución:**
> *   **Suma de positivos:** $65 + 12 = 77$.
> *   **Suma de negativos:** $50 + 30 = 80$ (el resultado es $-80$).
> *   **Operación final:** $-80 + 77$.
> 
> **Lógica de signos:** Como el valor absoluto de $80$ es mayor que el de $77$, el resultado conserva el signo del número mayor (negativo). Por lo tanto: $-80 + 77 = -3$.
> 
> **Interpretación:** El resultado es $-3$. La respuesta final es: **El explorador se encuentra a 3 metros bajo el nivel del suelo.**

> [!example] Ejemplo B — Extracción de cápsula submarina
> **Problema:** Una cápsula de exploración está a $150\text{ m}$ bajo el nivel del mar. Un barco la eleva a una velocidad de $25\text{ m/min}$. ¿A qué profundidad estará tras $5$ minutos?
> 
> **Planteamiento:**
> 1.  Profundidad inicial: $-150$.
> 2.  Cálculo de ascenso: $25\text{ m/min} \times 5\text{ minutos} = 125\text{ m}$ (valor positivo porque sube).
> 
> **Operación:**
> $-150 + 125 = -25$
> 
> **Interpretación:** El resultado $-25$ se interpreta como $25\text{ m}$ bajo el nivel del mar porque el signo negativo representa la ubicación por debajo del punto de referencia cero. La respuesta es: **La cápsula se encuentra a 25 metros bajo el nivel del mar.**

## 4. EJERCICIOS DE REPASO

> [!abstract] 🟢 Fácil (Buzo)
> Un buzo se encuentra a $15\text{ m}$ bajo la superficie del mar. Durante su búsqueda, asciende $9\text{ m}$ y luego desciende $30\text{ m}$ para alcanzar un objeto. ¿A qué profundidad se encontraba el objeto?

> [!abstract] 🟡 Medio (Alpinistas)
> Un grupo de alpinistas inicia a $4,500\text{ m}$ sobre el nivel del mar. Suben $800\text{ m}$, bajan $300\text{ m}$, bajan $50\text{ m}$ para buscar agua, suben $600\text{ m}$ y suben $430\text{ m}$ para llegar a la cima. ¿A qué altura se encuentra la cima?

> [!abstract] 🔴 Avanzado (Historia)
> El filósofo Sócrates nació en el año $470\text{ A.C.}$ y murió en el año $399\text{ A.C.}$ Utilizando la fórmula $\text{Año final} - \text{Año inicial}$, calcule cuántos años vivió.
> 
> **Pista para la resolución:**
> Aplique la interacción de signos con el año de nacimiento (valor inicial):
> $(-399) - (-470) \rightarrow -399 + 470 = 71$

## 5. CONSEJO DE ESTUDIO

> [!tip] 💡 Estrategia de agrupación
> Para evitar errores de signo en problemas con múltiples movimientos o variaciones, utilice la estrategia de **agrupación de términos**: sume todos los números positivos por un lado y todos los negativos por otro. Esta técnica está diseñada para **reducir la carga cognitiva** y simplificar la operación final a una sola resta entre dos valores, minimizando significativamente la probabilidad de fallos en el proceso de cálculo.
