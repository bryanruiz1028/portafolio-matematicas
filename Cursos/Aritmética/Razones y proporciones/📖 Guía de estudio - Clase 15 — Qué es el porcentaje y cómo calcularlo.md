# 📖 Guía de estudio — Clase 15: ¿Qué es el PORCENTAJE?

¡Hola! Para dominar los porcentajes, primero debemos entender qué estamos haciendo realmente. No se trata solo de mover comas, sino de entender cómo comparamos cantidades. ¡Vamos a darle con toda la energía!

> [!info] **📌 Conceptos clave**
> *   **¿Qué es el porcentaje?** Es una **razón** que utiliza el número $100$ como punto de referencia. Básicamente, nos indica cuántas partes tomamos por cada $100$ unidades.
> *   **Razón vs. Fracción:** El "profe Alex" prefiere el término **razón**. ¿Por qué? Porque las fracciones, estrictamente, deben usar números enteros. En cambio, una razón permite decimales (como $12,5$ partes de cada $100$), lo cual es muy común en los porcentajes.
> *   **El símbolo $\%$:** Este símbolo es una forma abreviada de escribir una división entre $100$. Verlo es como leer "dividido por cien".
> *   **Utilidad:** Los usamos para entender el mundo: desde cuántas mujeres hay en un grupo hasta los descuentos en tu tienda favorita o las estadísticas de un partido.

## 📐 Fórmulas y definiciones importantes

| Término | Definición / Fórmula |
| :--- | :--- |
| **Porcentaje** | Es una razón que indica una cantidad de cada $100$ unidades (denominador $100$). |
| **De $\%$ a Decimal** | Se quita el símbolo $\%$ y se corre la coma decimal **$2$ lugares hacia la izquierda**. |
| **De Decimal a $\%$** | Se multiplica el número por $100$, corriendo la coma **$2$ lugares hacia la derecha** y agregando el símbolo $\%$. |
| **Cálculo de Porcentaje** | Para hallar el porcentaje de un número usamos: $\frac{\text{porcentaje}}{100} \times \text{número}$ |

## 📝 Ejemplos resueltos paso a paso

```ad-example
title: Ejemplo A — Conversión de Porcentaje a Decimal
**Problema:** Convierte $15\%$ a número decimal.

1.  **Identifica la coma:** En el número entero $15$, la coma decimal está "invisible" al final: $15,$.
2.  **Aplica la regla:** Como queremos pasar de porcentaje a decimal, dividimos entre $100$ moviendo la coma dos lugares a la izquierda.
3.  **Movimiento:**
    *   Lugar 1: $1,5$
    *   Lugar 2: $,15$
4.  **Resultado:** Agregamos un cero inicial para que la coma no quede sola. El resultado es $0,15$. 
*(Nota: Si el resultado fuera $0,20$, siguiendo la lógica de Alex, eliminaríamos el cero final para dejarlo como $0,2$).*
```

```ad-example
title: Ejemplo B — Aplicación de Cálculo con Dinero ($USD)
**Problema:** Hallar el $20\%$ de $\$500$ USD.

1.  **Planteamiento:** Escribimos el porcentaje como razón y multiplicamos:
    $$\frac{20}{100} \times 500$$
2.  **Simplificación de ceros:** Para facilitar el cálculo mental, "tachamos" dos ceros del denominador ($100$) y dos ceros del número ($500$). Esto es como dividir arriba y abajo entre $100$.
    $$\frac{20}{1} \times 5$$
3.  **Operación final:** Multiplicamos los números que quedaron: $20 \times 5 = 100$.
4.  **Resultado:** El $20\%$ de $\$500$ USD es **$\$100$ USD**.
```

> [!tip] **💡 Consejo de estudio**
> ¡No te limites a memorizar! Si dudas hacia dónde va la coma, recuerda la lógica:
> *   **De Porcentaje a Decimal:** Estás **dividiendo** por $100$, por lo que el número debe hacerse más pequeño (la coma va a la **izquierda** $\leftarrow$).
> *   **De Decimal a Porcentaje:** Estás **multiplicando** por $100$, por lo que el número debe hacerse más grande (la coma va a la **derecha** $\rightarrow$).
>
> Entender el "porqué" de la operación es lo que te hará un maestro en matemáticas. ¡Tú puedes!
