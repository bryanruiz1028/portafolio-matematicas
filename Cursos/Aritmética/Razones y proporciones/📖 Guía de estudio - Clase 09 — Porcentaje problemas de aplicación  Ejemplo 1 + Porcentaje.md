# 📖 Guía de estudio — Clase 09: Porcentaje problemas de aplicación

> [!info] 📌 Conceptos clave
> El porcentaje es una aplicación de la **regla de tres simple directa**. Para resolver con éxito cualquier ejercicio, el primer paso es identificar el **$100\%$**, que siempre representa el valor total, el costo inicial o la base de referencia. Durante el análisis, es fundamental determinar si el problema solicita calcular el monto de una variación (descuento o ganancia) o el **monto total** tras aplicar dicha variación.

---

### Fórmulas y Definiciones Importantes

| Término | Definición |
| :--- | :--- |
| **$100\%$** | Representa la totalidad de una cantidad, el precio original de un producto o la base de comparación. |
| **Regla de Tres Directa** | Método para hallar una incógnita ($x$) multiplicando los valores conocidos en diagonal y dividiendo por el valor restante. |
| **Valor Neto / IVA** | El **Valor Neto** es el precio de un producto antes de impuestos. El **IVA** (Impuesto al Valor Agregado) es un recargo (ej. $19\%$) que se suma al neto. <br> **Fórmula:** $Valor\ Neto + IVA = Precio\ de\ Venta$. |

---

### Ejemplos Resueltos Adicionales

```ad-example
**Ejemplo A: Hallar el porcentaje de descuento**
**Problema:** Un televisor costaba $\$800$ y hoy cuesta $\$100$ menos. ¿Cuál es el porcentaje de descuento?

1. **Planteamiento de la tabla:**
| Porcentaje | Dinero ($) |
| :---: | :---: |
| $100\%$ | $800$ |
| $x$ | $100$ |

2. **Cálculo con Regla de Tres:**
$x = \frac{100 \times 100}{800}$

*   **Nota didáctica:** En el numerador, el primer $100$ representa el porcentaje base ($100\%$) y el segundo $100$ representa la cantidad de dinero descontada ($\$100$).

3. **Resolución:**
$x = \frac{10000}{800} = 12.5$

**Respuesta:** El porcentaje de descuento aplicado es del **$12.5\%$**.
```

```ad-example
**Ejemplo B: Aplicación de ganancia sobre el costo**
**Problema:** Se compra un artículo en $\$5,000$ y se desea obtener una ganancia del $15\%$. ¿Cuál debe ser el precio de venta?

1. **Paso 1: Cálculo de la ganancia en dinero ($x$)**
$x = \frac{15 \times 5000}{100} = 750$

2. **Paso 2: Identificación de valores**
La ganancia obtenida es de **$\$750$**.

3. **Paso 3: Cálculo del precio de venta**
Se suma la ganancia al costo inicial:
$\$5,000 + \$750 = \$5,750$

**Respuesta:** El precio de venta final debe ser **$\$5,750$**.
```

### Ejercicios de Repaso
¡Mucho ánimo! Intenta resolver estos retos para fortalecer tu razonamiento lógico.

```ad-abstract
**🟢 Nivel Fácil**
Un computador tiene un precio normal de **$\$1,200$**. Si hoy se vende con un **$30\%$ de descuento**, ¿cuánto se debe pagar por él al final?
*(Sugerencia: Puedes calcular el $30\%$ y restarlo, o hallar directamente el $70\%$ del valor inicial).*
```

```ad-abstract
**🟡 Nivel Medio**
Un comerciante compra una caja de dulces por **$\$7,500$** y decide venderla por **$\$9,000$**. Determina el **porcentaje de ganancia** obtenido en esta operación.
*(Pista: Primero resta el valor de venta menos el costo para saber cuánto dinero ganó exactamente).*
```

```ad-abstract
**🔴 Nivel Avanzado**
Silvia compra una impresora por un valor de **$\$3,200$** (precio que ya incluye el IVA). Si la tienda le otorga un **descuento del $25\%$** por aniversario, ¿cuál es el monto total que Silvia debe pagar?
*(Reto: Intenta plantear una sola operación para encontrar el pago final directamente).*
```

---

> [!tip] 💡 Consejo de estudio
> Para ganar velocidad en tus exámenes, **analiza la pregunta antes de operar**. Si te piden el "valor final a pagar" con un $10\%$ de descuento, es mucho más eficiente calcular directamente el **$90\%$** del valor original. Así, obtienes el resultado en un solo paso y evitas posibles errores en la resta final.
