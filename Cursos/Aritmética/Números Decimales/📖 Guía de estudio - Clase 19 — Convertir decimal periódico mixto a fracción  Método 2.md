# Guía de Estudio: Conversión de Decimales Periódicos Mixtos a Fracciones

Esta guía ha sido diseñada para estudiantes de Básica Superior con el objetivo de facilitar la comprensión del proceso matemático necesario para transformar un número decimal periódico mixto en una fracción.

---

## 1. Conceptos Fundamentales

Para dominar este tema, es esencial entender qué estamos convirtiendo.

### ¿Qué es un número decimal periódico mixto?
Es un número decimal cuyas cifras después de la coma están compuestas por dos partes:
1.  **Parte decimal no periódica:** Cifras que no se repiten (llamadas a veces anteperíodo).
2.  **Parte periódica:** Una o varias cifras que se repiten infinitamente. Se identifica con una línea sobre los números que se repiten.

**Ejemplos del texto:**
*   $1,1\bar{6}$: El $1$ es la parte no periódica y el $6$ es el período ($1,1666...$).
*   $23,7\bar{32}$: El $7$ no es periódico y el $32$ es el período ($23,7323232...$).

---

## 2. Método de Conversión Paso a Paso

El proceso se basa en el uso de ecuaciones para eliminar la parte periódica. Sigue estos cinco pasos:

| Paso | Acción | Descripción |
| :--- | :--- | :--- |
| **1** | **Igualar a $x$** | Se escribe la ecuación inicial: $x = \text{número decimal}$. |
| **2** | **Convertir a periódico puro** | Se multiplica toda la ecuación por una potencia de 10 ($10, 100, 1000$) para correr la coma hasta justo **antes** de donde empieza el período. |
| **3** | **Correr el período** | Se vuelve a multiplicar la ecuación obtenida por otra potencia de 10 para correr la coma hasta que cubra **una vez el período completo**. |
| **4** | **Restar ecuaciones** | A la ecuación del paso 3 se le resta la ecuación del paso 2. Esto elimina la parte periódica. |
| **5** | **Resolver y simplificar** | Se despeja la $x$ y se reduce la fracción a su mínima expresión. |

---

## 3. Ejemplo Práctico: Convertir $1,1\bar{6}$

1.  **Igualar:** $x = 1,1\bar{6}$
2.  **Mover coma antes del período:** Como hay una cifra no periódica (el $1$), multiplicamos por $10$.
    *   $10x = 11,\bar{6}$ (Ahora es un periódico puro).
3.  **Mover la coma después del período:** El período tiene una cifra ($6$), multiplicamos la ecuación anterior por $10$ nuevamente.
    *   $100x = 116,\bar{6}$
4.  **Restar:**
    *   $100x = 116,\bar{6}$
    *   $- 10x = 11,\bar{6}$
    *   Resultado: $90x = 105$ (la parte periódica desaparece).
5.  **Resolver y simplificar:**
    *   $x = \frac{105}{90}$
    *   Simplificando (tercera y luego quinta): $\frac{105}{90} \rightarrow \frac{35}{30} \rightarrow \frac{7}{6}$
    *   **Resultado final:** $7/6$

## 6. Glosario de Términos Clave

*   **Decimal Periódico Mixto:** Número decimal que tiene una parte que no se repite y otra que sí se repite infinitamente después de la coma.
*   **Decimal Periódico Puro:** Número decimal donde el período comienza inmediatamente después de la coma.
*   **Ecuación:** Igualdad matemática entre dos expresiones que contiene una o más incógnitas (en este caso, $x$).
*   **Período:** La cifra o grupo de cifras que se repiten de forma infinita en un número decimal.
*   **Potencia de 10:** Números como $10, 100, 1000$, etc., que se obtienen multiplicando el $10$ por sí mismo y que sirven para desplazar la coma decimal hacia la derecha.
*   **Simplificar:** Proceso de reducir una fracción a su forma más pequeña dividiendo el numerador y el denominador por el mismo número.
