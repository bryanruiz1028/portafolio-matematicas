# Conversión de Decimales Periódicos Mixtos a Fracciones: Guía de Estudio

Esta guía proporciona una explicación detallada sobre el procedimiento matemático para transformar números decimales periódicos mixtos en fracciones, basándose en los métodos de resolución y simplificación expuestos en el material de análisis.

---

## 1. Conceptos Fundamentales

Un **número decimal periódico mixto** es aquel que, después de la coma (o punto) decimal, presenta dos partes diferenciadas:
1.  **Parte no periódica (Anteperíodo):** Una o más cifras que no se repiten.
2.  **Parte periódica (Período):** Una o más cifras que se repiten infinitamente. Este período se identifica usualmente con una línea horizontal sobre los números que se repiten.

**Ejemplos de estructura:**
*   $7,0\bar{5}$: El $0$ es la parte no periódica y el $5$ es el período ($7,0555...$).
*   $3,2\overline{45}$: El $2$ es la parte no periódica y el $45$ es el período ($3,2454545...$).

---

## 2. Procedimiento de Conversión

Para convertir estos decimales a una fracción, se sigue una regla dividida en dos partes: el numerador y el denominador.

### A. Construcción del Numerador
Se realiza una resta siguiendo estos pasos:
1.  **Número completo:** Se escribe el número decimal como si fuera un entero, eliminando la coma y la línea del período.
2.  **Resta de la parte no periódica:** A ese número se le resta la cifra formada por todo lo que está antes del período (incluyendo la parte entera), sin la coma.

### B. Construcción del Denominador
El denominador se forma combinando nueves ($9$) y ceros ($0$):
1.  **Tantos nueves como cifras tenga el período:** Si el período tiene una cifra, se pone un $9$; si tiene dos, se ponen dos $99$, y así sucesivamente.
2.  **Tantos ceros como cifras tenga la parte decimal no periódica:** Se observan únicamente las cifras decimales que no se repiten. Si hay una cifra, se agrega un $0$; si hay dos, se agregan dos $00$.

> **Regla mnemotécnica:** Las flechas de conteo para nueves (período) y ceros (decimal no periódico) no deben cruzarse al identificar las cifras en el número original.

## 3. Ejemplos de Aplicación y Simplificación

### Ejemplo 1: $7,0\bar{5}$
*   **Numerador:** $705 - 70 = 635$
*   **Denominador:** $9$ (una cifra periódica) y $0$ (una cifra decimal no periódica) = $90$
*   **Fracción resultante:** $\frac{635}{90}$
*   **Simplificación:** Sacando quinta parte $\rightarrow \frac{127}{18}$

### Ejemplo 2: $3,2\overline{45}$
*   **Numerador:** $3245 - 32 = 3213$
*   **Denominador:** $99$ (dos cifras periódicas) y $0$ (una cifra decimal no periódica) = $990$
*   **Fracción resultante:** $\frac{3213}{990}$
*   **Simplificación:** Sacando tercera dos veces $\rightarrow \frac{357}{110}$

### El Caso Especial del Período 9: $8,75\bar{9}$
Cuando el período es $9$, el resultado de la fracción tiende a aproximar la cifra anterior.
*   **Operación:** $\frac{8759 - 875}{900} = \frac{7884}{900}$
*   **Resultado simplificado:** $\frac{219}{25}$
*   **Nota:** Al realizar la división $219 \div 25$, el resultado es $8,76$. Esto sucede porque el $9$ periódico aproxima la cifra decimal anterior sumándole uno.

## 5. Glosario de Términos Clave

| Término | Definición |
| :--- | :--- |
| **Decimal Periódico Mixto** | Número decimal que tiene una parte que no se repite seguida de una parte que se repite infinitamente. |
| **Período** | La cifra o grupo de cifras que se repiten indefinidamente en un número decimal. |
| **Anteperíodo** | La cifra o grupo de cifras decimales que se encuentran entre la coma y el comienzo del período. |
| **Simplificación** | Proceso de reducir una fracción a su forma más simple dividiendo el numerador y el denominador por el mismo número. |
| **Numerador** | La parte superior de una fracción, que en este proceso representa la diferencia entre el número total y la parte no periódica. |
| **Denominador** | La parte inferior de una fracción, compuesta por nueves y ceros según la cantidad de cifras del período y el anteperíodo. |
