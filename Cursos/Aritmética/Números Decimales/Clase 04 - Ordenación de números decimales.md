## 1. ¿Por qué es importante esta clase?

> [!info] 🌍 Relevancia y aplicaciones
> ¿Alguna vez te has preguntado quién ganó una carrera por apenas un pestañeo? Saber ordenar decimales te permite entender quién tiene más, quién es más rápido o qué medida es más exacta. ¡Es la clave de la precisión!
> - 💵 **Dólares:** Para comparar precios en el súper y elegir la oferta que de verdad te hace ahorrar.
> - 🏗️ **Construcción:** Para que los planos sean perfectos y las piezas encajen sin que sobre ni un milímetro.
> - 📊 **Estadísticas:** Para diferenciar resultados deportivos donde una milésima decide quién se lleva la medalla de oro.

## 2. Concepto clave

> [!note] 📌 ¿Qué es Order of decimal numbers?
> Un número decimal tiene dos mundos separados: la **parte entera** (a la izquierda) y la **parte decimal** (a la derecha).
> 
> **¡Ojo aquí!** 
> 1. Según tu país, se puede usar una **coma (,)** o un **punto (.)** para separarlos. ¡Ambos son válidos!
> 2. **La coma invisible:** Los números enteros (como el 52) parecen no tener decimales, pero en realidad tienen una coma escondida **al puro final** (52,). Esto es vital para compararlos con otros números.

> [!warning] ⚠️ ¡No te dejes engañar!
> ❌ **Error común:** Pensar que $7,39$ es más grande que $7,4$ solo porque "39 es más que 4".
> ✅ **Lo correcto:** Debes comparar cifra por cifra. Si miras las décimas, el **4** le gana al **3**. Por lo tanto, ¡$7,4$ es el mayor!

> [!tip] 💡 Truco maestro: Rellenar con ceros
> Si te confunde ver números con diferentes largos, simplemente **añade ceros a la derecha** del número más corto hasta que ambos tengan la misma cantidad de cifras. 
> *Ejemplo:* Comparar $7,39$ contra $7,40$ es mucho más fácil, ¿verdad?

## 3. Procedimiento paso a paso

Sigue esta lógica del Profe Alex para no fallar nunca:

```text
PASO 1: Compara la parte entera. Si una es más grande, ¡listo! Ese número es el mayor.
PASO 2: Si las partes enteras son iguales, mira las décimas (la primera cifra tras la coma).
PASO 3: Si siguen iguales, revisa las centésimas (la segunda cifra).
PASO 4: Continúa cifra por cifra (milésimas, etc.) o rellena con ceros para igualar el largo.
```

## 4. Ejemplos prácticos

```ad-example
title: Ejemplo 1: ¡La parte entera manda!
Compara **20,3** vs **23,1**.
- Parte entera A: **20**
- Parte entera B: **23**
**¡Veredicto!:** Como la parte entera es distinta, **ya no necesitas mirar los decimales**. El número con la parte entera más grande gana automáticamente.
**Resultado:** $20,3 < 23,1$
```

```ad-example
title: Ejemplo 2: Cifra por cifra (Décimas y Centésimas)
Compara **20,31** vs **20,35**.
1. Enteros: $20 = 20$ (Empate)
2. Décimas: $3 = 3$ (Empate)
3. Centésimas: Comparas el **1** contra el **5**.
**¡Veredicto!:** El 5 es mayor.
**Resultado:** $20,31 < 20,35$
```

```ad-example
title: Ejemplo 3: Diferente longitud (Usa el truco)
Compara **7,39** vs **7,4**.
Alineamos y rellenamos con el cero invisible:
$7, \mathbf{3} 9$
$7, \mathbf{4} 0$
**¡Veredicto!:** Al mirar las décimas, el 4 es mayor que el 3.
**Resultado:** $7,39 < 7,4$
```

```ad-example
title: Ejemplo 4: Aplicación en dinero ($USD)
En una gasolinera, el precio A es **$0,70** y el precio B es **$0,705**.
¿Cuál es más caro?
- Precio A: $0,70\mathbf{0}$
- Precio B: $0,70\mathbf{5}$
**¡Veredicto!:** El precio B es mayor por 5 milésimas.
**Resultado:** El producto B es más caro.
```

## 5. Ejercicios para el estudiante

```ad-abstract
title: 🟢 Nivel Fácil: Partes enteras diferentes
Escribe $>$ (mayor que) o $<$ (menor que) según corresponda:
1. $5,2$ \_\_\_ $3,8$
2. $4,1$ \_\_\_ $3,9$
3. $12,1$ \_\_\_ $5,7$
4. $154$ \_\_\_ $95,2$ (Recuerda la coma invisible en el 154)
```

```ad-abstract
title: 🟡 Nivel Medio: Longitudes tramposas
Escribe $>$, $<$ o $=$ según corresponda:
1. $8,1$ \_\_\_ $8,09$
2. $8,71$ \_\_\_ $8,695$
3. $5,42$ \_\_\_ $5,43$
4. $7,2$ \_\_\_ $7,20$
```

```ad-abstract
title: 🔴 Nivel Avanzado: El gran desafío de orden
Ordena los siguientes números de **menor a mayor**. ¡Ten cuidado con los enteros puros!
`52.25, 50, 53.1, 52.072, 51.42, 53, 52.7, 52.9`
```

```ad-success
title: ✅ Respuestas (para el docente)
**Fácil:** 1. $>$ | 2. $>$ | 3. $>$ | 4. $>$
**Medio:** 1. $>$ | 2. $>$ | 3. $<$ | 4. $=$
**Avanzado:** $50 < 51,42 < 52,072 < 52,25 < 52,7 < 52,9 < 53 < 53,1$
```

## 6. Mini-prueba de autoevaluación

```ad-question
title: Pregunta 1 (Teoría)
¿Qué es lo primero que debes comparar en un número decimal para saber si es mayor?
**Respuesta:** Siempre, siempre, la parte entera.
```

```ad-question
title: Pregunta 2 (Práctica)
¿Qué símbolo hace que esta expresión sea real? $8,71$ \_\_\_\_ $8,695$
**Respuesta:** El símbolo **$>$** (mayor que), porque 7 décimas ganan a 6 décimas.
```

```ad-question
title: Pregunta 3 (Aplicación real)
Si el producto A cuesta $\$12,1$ y el B cuesta $\$12,09$, ¿por cuál pagarías más dinero?
**Respuesta:** Por el producto A, ya que $\$12,10$ es mayor que $\$12,09$.
```
