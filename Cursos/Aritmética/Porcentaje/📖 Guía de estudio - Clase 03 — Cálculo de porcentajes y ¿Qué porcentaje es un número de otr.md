# 📖 Guía de estudio — Clase 03: Qué porcentaje es un número de otro

## SECCIÓN 1: CONCEPTOS CLAVE

> [!info] 📌 Conceptos fundamentales
> * **El 100% como Total:** En cualquier situación de proporcionalidad, el valor de referencia inicial o la cantidad completa siempre se define como el $100\%$.
> * **Regla de Tres Simple Directa:** Es el **método formal** para resolver porcentajes cuando conocemos tres datos y buscamos una incógnita ($x$). Es ideal cuando los números no terminan en cero o son complejos.
> * **Técnicas de Desplazamiento:** Para calcular el $10\%$ o el $1\%$, usamos la lógica de "correr la coma" hacia la izquierda. Si un número no tiene punto decimal visible (como $42$), recuerda que el punto está al final ($42.0$).
> * **Ubicación de la Incógnita ($x$):** Cuando el problema pregunta "¿Qué porcentaje es...?", la incógnita siempre debe colocarse en la columna de los porcentajes.

## SECCIÓN 2: FÓRMULAS Y DEFINICIONES

| Término | Definición / Fórmula |
| :--- | :--- |
| **$100\%$ (Total)** | El valor base o cantidad total de referencia. |
| **$10\%$** | Se obtiene dividiendo el total entre $10$ (correr el punto decimal una posición a la izquierda). |
| **$5\%$** | Se define como **la mitad del valor del $10\%$**. Es un puente clave para el cálculo mental. |
| **$1\%$** | Se obtiene dividiendo el total entre $100$ (correr el punto decimal dos posiciones a la izquierda). |
| **Regla de Tres** | Método formal: $x = \frac{(\text{valor parcial} \cdot 100)}{\text{valor total}}$ |

## SECCIÓN 3: ESTRATEGIAS DE CÁLCULO MENTAL

Para calcular porcentajes sin usar calculadora, aplicamos la **descomposición**. Esto consiste en ver un porcentaje complejo como la suma de porcentajes base ($50\%, 10\%, 5\%, 1\%$).

**Ejemplo de lógica mental:**
* Para obtener el $21\%$, pensamos en: $10\% + 10\% + 1\%$.
* Para obtener el $56\%$, pensamos en: $50\% + 5\% + 1\%$.
Recuerda: si ya tienes el $10\%$, el $5\%$ es simplemente su mitad. ¡Es mucho más rápido que multiplicar!

## SECCIÓN 4: EJEMPLOS RESUELTOS ADICIONALES

```ad-example
title: Ejemplo A — Caso básico (Hallar porcentaje de un número)
**Problema:** Calcular el $21\%$ de $2000$.

**Estrategia mental (Desglose):**
1. **$10\%$ de $2000$:** Quitamos un cero $\longrightarrow 200$.
2. **Otro $10\%$:** Sumamos otros $\longrightarrow 200$.
3. **$1\%$ de $2000$:** Quitamos dos ceros $\longrightarrow 20$.

**Operación final:** $200 + 200 + 20 = 420$.
✅ **Resultado:** El $21\%$ de $2000$ es **$420$**.
```

```ad-example
title: Ejemplo B — Caso de aplicación real $USD (Qué porcentaje es un número de otro)
**Problema:** ¿Qué porcentaje representa $35$ de un total de $130$?

**Planteamiento (Regla de tres):**
- $130 \longrightarrow 100\%$
- $35 \longrightarrow x$

**Operación y Simplificación de ceros:**
$x = \frac{35 \cdot 100}{130} = \frac{3500}{130}$

Aplicamos el consejo de "quitar ceros" (arriba y abajo):
$\frac{350\cancel{0}}{13\cancel{0}} = \frac{350}{13} \approx 26.9$

✅ **Resultado:** $35$ es el **$26.9\%$** de $130$.
```

## SECCIÓN 6: CONSEJO DE ESTUDIO

> [!tip] 💡 La magia de la simplificación
> Como enseña el **Profe Alex**, para facilitar los cálculos manuales en la regla de tres, siempre intenta **simplificar ceros arriba y abajo** en la fracción antes de dividir. Por ejemplo, en $\frac{3500}{130}$, eliminar un cero en ambos lados transforma una división difícil en una mucho más sencilla: $\frac{350}{13}$. ¡Menos ceros, menos errores!
