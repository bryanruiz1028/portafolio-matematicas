### 2. Resolución Detallada de Ejercicios

**Q9 (Lanzamiento de dado):**
*   **Espacio Muestral ($\Omega$):** {1, 2, 3, 4, 5, 6} -> $n=6$
*   **Casos favorables (A < 4):** {1, 2, 3} -> $n=3$
*   **Cálculo:** $P(A) = 3/6 = 1/2 = 0.5$
*   **Resultado:** 50% de probabilidad.

**Q10 (Bono Gerencial - Teorema de Probabilidad Total):**
*   **Datos:** 
    *   $P(\text{Abrir}) = 0.6$
    *   $P(\text{No Abrir}) = 0.4$
    *   $P(\text{Gerente}|\text{Abrir}) = 0.7$
    *   $P(\text{Gerente}|\text{No Abrir}) = 0.1$
*   **Fórmula:** $P(G) = [P(A) \times P(G|A)] + [P(A') \times P(G|A')]$
*   **Cálculo:** $(0.6 \times 0.7) + (0.4 \times 0.1) = 0.42 + 0.04 = 0.46$
*   **Resultado:** 46% de probabilidad de éxito.
