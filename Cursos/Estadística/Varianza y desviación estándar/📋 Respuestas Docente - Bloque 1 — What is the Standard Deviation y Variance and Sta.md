## Solucionario Detallado

**SECCIÓN I: Selección Múltiple**
*   **Q1: b)** Según la base pedagógica de Profe Alex, la desviación estándar mide qué tan separados están los datos del promedio.
*   **Q2: a)** Es la distinción técnica clave: la población usa el total de datos ($n$), mientras que la muestra aplica una corrección de grados de libertad ($n-1$).
*   **Q3: c)** Una desviación mayor indica que los datos están más alejados de la media, aumentando la heterogeneidad.

**SECCIÓN II: Verdadero o Falso**
*   **Q4: V.** La varianza eleva las diferencias al cuadrado, por lo que su unidad también queda al cuadrado.
*   **Q5: V.** Operacionalmente, la desviación estándar es la raíz de la varianza para volver a la unidad original.
*   **Q6: V.** Basado en la definición de Profe Alex, la desviación media utiliza valores absolutos $|x - \bar{x}|$ para evitar que las diferencias se cancelen entre sí.

**SECCIÓN III: Completar Espacios**
*   **Q7: 25% (a), heterogéneos (d), dispersión (e).** Justificación: Según la fuente, el umbral del 25% es el estándar pedagógico para diferenciar una distribución homogénea de una heterogénea.
*   **Q8: homogéneos (a), desviación estándar (b), promedio (c).** Justificación: El coeficiente de variación normaliza la desviación estándar respecto a la media.

**SECCIÓN IV: Relacionar Columnas**
*   $\sigma^2$ -> C (Varianza poblacional)
*   $s$ -> D (Desviación estándar muestral)
*   $\mu$ -> B (Promedio poblacional - Letra griega Mu)
*   $\bar{x}$ -> A (Promedio muestral)

**SECCIÓN V: Resolver Ejercicio (Población)**
1.  **Promedio ($\mu$):** $(5 + 6 + 7) / 3 = 6$ años.
2.  **Varianza ($\sigma^2$):** 
    *   $[(5-6)^2 + (6-6)^2 + (7-6)^2] / 3$
    *   $[1 + 0 + 1] / 3 = 2 / 3 = 0.67$ años².
3.  **Desviación Estándar ($\sigma$):** $\sqrt{0.67} = 0.82$ años.

**SECCIÓN VI: Aplicación con USD (Muestra)**
1.  **Promedio ($\bar{x}$):** $(10 + 15 + 20) / 3 = 15$ USD.
2.  **Varianza ($s^2$):** 
    *   $[(10-15)^2 + (15-15)^2 + (20-15)^2] / (3-1)$
    *   $[25 + 0 + 25] / 2 = 50 / 2 = 25$ USD².
3.  **Desviación Estándar ($s$):** $\sqrt{25} = 5$ USD.
