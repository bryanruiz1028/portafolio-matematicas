# 📖 Guía de estudio — Clase 04: Volumen de un cilindro

> [!info] 📌 Conceptos clave
> El volumen de un cilindro es una medida de la capacidad de este sólido geométrico. Como tu tutor, te presento los puntos fundamentales que debes dominar:
> - **Definición:** Un cilindro es un sólido o figura geométrica caracterizado por tener una base circular.
> - **Cálculo general:** El volumen se obtiene multiplicando el área de la base circular por la altura del sólido.
> - **Importancia del radio ($r$):** El radio es el elemento esencial para determinar el área del círculo de la base.
> - **Unidades de medida:** Es vital distinguir entre unidades cuadradas ($cm^2, m^2$) para el área, y unidades cúbicas ($cm^3, m^3$) para el volumen.
> - **Consistencia de unidades:** Antes de operar, asegúrate de que el radio y la altura estén en la misma unidad (ambos en metros o ambos en centímetros).
> - **Valor de $\pi$:** Para todos nuestros ejercicios, utilizaremos el valor convencional de **3.1416**.

## Fórmulas y definiciones importantes

| Término | Definición / Fórmula |
| :--- | :--- |
| **Cilindro** | Sólido geométrico con una base en forma de círculo y una altura determinada. |
| **Radio ($r$)** | Distancia desde el centro del círculo de la base hasta el borde. *Nota: Si el problema te da el diámetro, recuerda dividirlo entre 2 para obtener el radio ($r = d/2$).* |
| **Volumen ($V$)** | Se calcula con $V = A_{base} \times h$. Si conoces el radio, la fórmula extendida es $V = \pi \times r^2 \times h$. |
| **$\pi$ (Pi)** | Valor numérico irracional que utilizaremos como **3.1416** para facilitar los cálculos. |

## Ejemplos resueltos adicionales

```ad-example
title: Ejemplo A — Cálculo con área de base conocida
**Enunciado:** Determina el volumen de un cilindro cuya base mide $20 cm^2$ y su altura es de $15 cm$.

**Instrucciones de resolución:**
1. **Identificar la fórmula:** Al tener el área de la base directamente, usamos $V = A_{base} \times h$.
2. **Sustituir valores:** $V = 20 cm^2 \times 15 cm$.
3. **Realizar la multiplicación:** $20 \times 15 = 300$.

✅ **Resultado:** El volumen es de **$300 cm^3$**. Recuerda que los exponentes de las unidades se suman ($cm^2 \times cm^1 = cm^3$).
```

```ad-example
title: Ejemplo B — Aplicación real con radio y costo
**Enunciado:** Se tiene un tanque cilíndrico de almacenamiento con un radio de $25 m$ y una altura de $13 m$. Si el costo de mantenimiento del espacio es de $0.50 USD por metro cúbico, ¿cuál es el volumen total y el costo asociado?

**Instrucciones de resolución:**
1. **Calcular el radio al cuadrado:** $25^2 = 25 \times 25 = 625$.
2. **Aplicar la fórmula completa:** $V = \pi \times r^2 \times h$.
   - $V = 3.1416 \times 625 m^2 \times 13 m$
   - $V = 25,525.5 m^3$
3. **Calcular el costo:** Multiplicamos el volumen por el precio unitario.
   - $25,525.5 \times 0.50 = 12,762.75$

✅ **Resultado:** El volumen total es de **$25,525.5 m^3$** y el costo de mantenimiento asciende a **$12,762.75 USD**.
```

## Ejercicios de repaso

```ad-abstract
title: 🟢 Fácil
1. Calcula el volumen si el área de la base es $10 cm^2$ y la altura es $5 cm$.
2. Calcula el volumen si el área de la base es $50 m^2$ y la altura es $2 m$.
3. ¿Cuál es el volumen de un cilindro con base de $12 cm^2$ y altura de $10 cm$?
```

```ad-abstract
title: 🟡 Medio
1. Calcula el volumen de un cilindro con radio $r = 5.3 cm$ y altura $h = 13.2 cm$.
2. Determina el volumen si el radio es $3 m$ y la altura es $10 m$.
3. Un cilindro tiene un radio de $4 cm$ y altura de $6 cm$. Halla su volumen.
```

```ad-abstract
title: 🔴 Avanzado — Aplicación con moneda
1. Un contenedor de aceite tiene un radio de $2 m$ y una altura de $5 m$. Si cada metro cúbico de aceite cuesta $120 USD, ¿cuál es el valor total del contenido del cilindro lleno?
2. Se desea llenar una piscina cilíndrica de $3 m$ de radio y $1.5 m$ de profundidad. Si el agua cuesta $2 USD por cada metro cúbico, ¿cuánto costará llenarla?
```

> [!tip] 💡 Consejo de estudio
> Para evitar errores en la calculadora, sigue siempre este orden: **primero eleva el radio al cuadrado**, luego multiplica por $\pi$ y finalmente por la altura. No olvides verificar que todas tus medidas estén en la misma unidad antes de empezar; si una está en metros y otra en centímetros, ¡debes convertirlas primero!
