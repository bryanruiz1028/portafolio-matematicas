# Guía de Estudio Rápida: Triángulos

### 1. Cuadro de Fórmulas
*   **Perímetro ($P$):** $a + b + c$
*   **Semiperímetro ($s$):** $(a + b + c) / 2$
*   **Área (Fórmula de Herón):** $\sqrt{s(s-a)(s-b)(s-c)}$
*   **Suma de Ángulos Internos:** Siempre $180°$.

### 2. Las 5 Características Críticas
1.  **Es un Polígono:** Figura plana, cerrada y de 3 lados rectos.
2.  **Vértices:** 3 puntos de unión (esquinas).
3.  **Identidad:** Se nombran con mayúsculas ($ABC$) o letras griegas ($\alpha, \beta, \theta$).
4.  **Apertura:** Tienen 3 ángulos internos.
5.  **Puntos No Colineales:** Los vértices nunca pueden estar en una sola línea recta.

### 3. Identificación Visual (Las "Marcas")
Para saber qué tipo de triángulo tienes sin usar regla:
- **Equilátero:** Busca **tres rayitas iguales** (una en cada lado). Significa que todo es igual (lados y ángulos de 60°).
- **Isósceles:** Busca **dos rayitas iguales**. Significa que tiene dos lados y dos ángulos iguales.
- **Dato Extra:** La altura solo divide la base exactamente a la mitad si ves estas marcas de igualdad.
